Adding and Removing Resources

WinAGI automatically keeps track of the in-game status of all resources that are being edited or previewed.

Resources that are NOT currently part of your game can easily be added.

Resources that ARE currently part of your game can easily be removed.

Adding Resources to a Game

If you are editing a new logic, picture, sound or view resource that is not currently in your game (maybe you created or imported it earlier without adding it to your game at that time), use the Add To Game command (or toolbar button) to add it to your game.

The Add New Resource dialog window is displayed:

On this dialog, you choose the resource number, the resource ID, and an optional description. The default ID is 'Logic#', 'Picture#', etc. where # is the desired resource number. The new resource will then be added to your game when you click OK.

 

NOTE: WORDS.TOK files and OBJECT files cannot be added in this manner. If you want to replace your existing OBJECT or WORDS.TOK file with one that is currently open, save the one that is open, close it, then import it. Click 'Yes' when asked if you want to replace your existing file.

 

Removing Resources from a Game

If you want to remove a logic, picture, sound or view resource from your game, use the Remove from Game command (or the toolbar button).

When you remove a resource, you will be asked if you want to export it before it is removed. Choose Yes if you want a copy of the resource saved to file.

If you remove a resource WITHOUT exporting it, it will be gone forever and cannot be recovered.

NOTE: WORDS.TOK and OBJECT files cannot be removed from a game. All games must have these two files.

You can replace these files with new ones, or with imported ones.

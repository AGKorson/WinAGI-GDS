AGI Power Pack Support

The AGI Power Pack is a set of logic scripts that exploit a buffer overflow bug in the set.string command to add enhancements and new capabilities to the MSDOS versions of AGI. Mouse support, 80 column text support, sound card and MIDI music support, full color text printing, support for extended characters and alternate code pages, and dozens of new commands that expand the options for game designers to create much more complex and capable games are just some of what the Power Pack offers.

Incorporating the Power Pack into a game is complex and should only be used by experienced AGI programmers. If you do want to program a game to use the Power Pack, you can configure WinAGI to provide support for the Power Pack features that affect various resource editors.

To enable Power Pack support, edit your game's WAG file, and add this property to the General section:

   PowerPack = True

When enabled, the following enhancements are available in WinAGI:

    • WORDS.TOK editor allows use of extended characters in words: in normal AGI, extended characters are not allowed as words because the parser only allows characters with values less than 128. The Power Pack modifies AGI so extended character words can be used. When Power Pack setting is enabled, WinAGI will allow extended characters to be used in words and will correctly format them so they will work in your Power Pack game.
    • Picture Editor Text Test mode supports 80 column width: use the Screen Size menu item (or press ALT+W) in test mode to toggle the between 40 and 80 columns. Also, all colors are usable for background colors.
    • Text Screen Editor supports 80 column width: use the Screen Size menu item (or press ALT+W) to toggle the between 40 and 80 columns.
    • Palette Editor allows copying PowerPack compatible palette string: you can right-click on the dialog background or press 'CTRL+C' to copy the current palette into a string value that can be pasted into logic source code in a format that is compatible with the Power Pack palette functions. You can also paste a Power Pack palette string to quickly set a custom palette.
    • Menu Editor adds an option to display the menu structure with 80 columns of text, and provides access to the Character Map to allow addition of extended characters to menu text.

Details on Power Pack features, and how to use the various Power Pack commands can be found in the Power Pack source code and demo game. These can be found on the AGI Power Pack Github page.

Previewing Views

When previewing views, you can cycle through all loops and cels in a view. Use the two spinner controls to step the cel or loop forwards or backwards. You can also use the following shortcut keys:

' ' (spacebar) - toggle play/pause buttons

A - previous cel

S - next cel

Q - previous loop

W - next loop

A scale control is provided to adjust the scale of the view while previewing it.

The cel's transparent color is shown on the preview toolbar. This is the color that will be invisible when the view is drawn on a picture. You can toggle between showing and hiding the transparent color by clicking the Show/Hide button.

You can change the background color in the View preview window by double-clicking on the background area (useful to get a feel for what a cel might look like in a game). Choose one of the standard AGI colors, or use the preview window default color.

Double-clicking on the view's cel will open it in a View Editor for editing.

Views can be aligned to the top, middle, bottom or right, center or left of the preview window using the alignment buttons on the preview toolbar. When cycling through cels of different sizes, aligning to the bottom will let you see the cels cycle more naturally than if they are aligned to the top.

Use the bottom toolbar to preview the view in motion. Choose a movement mode from the dropdown box (normal, end of loop, reverse loop). Adjust the speed using the slider. Press the play button to start the animation, and the stop button to stop animation.

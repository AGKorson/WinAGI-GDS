Template Games

WinAGI allows you to create new games based on a template, so you don't have to re-create boiler plate game resources. Template games must be located in the Templates directory of the WinAGI program directory (usually C:\Program Files\WinAGI). Template games must be in the current WinAGI format; older versions must be opened and upgraded to be used as a template.

Creating a New Game from a Template

To create a new game based on a template, select From Template on the New Game menu. The template selection form will be shown, which allows you to choose from available templates:

Select the desired template from the list. The interpreter version used by the selected template is shown as well as the template game description. Click OK to use the template. The Game Properties window is then displayed to allow you to set the properties for your new game.

Creating and Editing Templates

Template games can be created and edited in WinAGI just like any other game.

If you want to make changes to a template game, open it with the Open Game command. (Remember that template games are stored in the Templates directory of the main program directory.) If you make changes to logics in the template, you should compile them so the logic resource in the volume files match the source code.

To remove templates, simply delete the game directory from the Templates directory.

WinAGI v2 Games as Templates

WinAGI Version 2 games cannot be used as templates. You must upgrade the game to current version (3) to use it as a template. Opening the game in WinAGI version 3 will automatically upgrade the game files.

Sound Settings

Default Scale

You can set the default scale factor for the staff display area when a sound editor is first opened. The minimum is one; the maximum is three.

 

Default Playback Mode

WinAGI provides two options for playing sounds: a PCJr sound chip emulator, and a MIDI conversion. (The PC Speaker emulator is currently not available; using the PCJr mode, with tracks 1, 2 and noise muted provides a close approximation.) When previewing sounds, or when opening a sound for editing, the Playback mode is set to the default value. You can still change the playback mode after the sound is loaded.

 

Default Track Settings for MIDI Playback

When sounds are created, or opened for the first time, they are given instrument settings that will apply when the playback mode is set to MIDI. With these settings you can assign the default instrument to play for each track and whether or not to mute the track. When a sound is opened for editing, you can make adjustments to instrument and mute properties as needed.

Reset All Sound Properties

Click this button to update all instrument and mute property values for all sounds in a game to match the currently selected default values.

 

NOTE: The Instrument and Mute properties are a WinAGI feature you can use to play back sounds in WinAGI; they are not used when you play a game using AGI or any other interpreter program.

 

Show Keyboard

Use this option to control whether the virtual keyboard is visible when a sound editor is opened.

 

Disable Keyboard Sound

Normally when a key is pressed on the virtual keyboard, the key's tone is played, providing audible feedback. In some situations the playback can be choppy, or you may not want the tone at all. Setting this option disables the virtual keyboard sound- no tone plays when keys are pressed on the virtual keyboard.

 

Show Notes as Bitmaps

When enabled, notes that have a duration that exactly matches a standard note length (whole, half, quarter, etc.) will be drawn as a bitmap representing the note. When disabled, all notes, regardless of note length, are drawn as rectangles, with the rectangle's width equal to the note's length.

 

Maximize Selected Track

When this option is enabled, the default state when opening a sound editor is to only display the staff for the the currently selected track. All other tracks are hidden, and all tracks' Visible property is ignored. This can be useful if you have a smaller display, such as a laptop, by reducing clutter as you are working on your sound resource.

Errors and Warnings List

The Errors and Warnings List displays all warnings and errors encountered when working on an AGI game. Each entry includes the error or warning number, a description, and the resource number and line number where the error or warning occurred, if applicable. If an error or warning is found in an included file, the name of the include file will be shown here to aid in finding the code that generated the error or warning.

The Warnings and Error List will display:

    • Game and Resource Errors
    • Game and Resource Warnings
    • Logic Source Decompiler Errors
    • Logic Source Decompiler Warnings
    • Logic Source Compiler Errors
    • Logic Source Compiler Warnings
    • Logic Source 'TODO' entries

The list will automatically display whenever a condition arises that adds a new item to the list. You can hide the list by using the Hide Warning List option on the Tools menu. You can re-display it from the same menu.

Filter and Sort Options

The list includes filtering and sorting capabilities to help manage your list. You can toggle the display of errors, warnings and TODO items by clicking on the count label at the top of the list. You can also change the scope of errors from the default (all items across all files) to only those that apply to the currently selected resource, all open resources, or only logics.

Right-click any column header to change the sort order of the list.

You can double-click on a TODO, error or warning entry for a logic source file to automatically open the logic source to the line at which the error or warning occurred.

For additional information related to an error or warning, select it then right-click and choose Help from the context menu (or press F1). The WinAGI help file will open to the appropriate page and display information on the selected error or warning.

Ignoring Individual Warnings

If you don't want to lower the Error Detection Level, but also don't want to see certain warnings, you can choose to ignore warnings individually. To ignore a warning, right click on the warning you want to ignore, then choose "Ignore Warning" from the popup menu. Note that this will cause WinAGI to not show this particular warning again for any logics you compile in any game. You can reset ignored warnings back to normal by using the Reset Message Box Warnings" option on the General tab of the Settings dialog.

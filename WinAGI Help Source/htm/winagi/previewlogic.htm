Previewing Logics

Logics are shown in the preview window in a simple text box. You can select text (maybe you want to copy some code for use in another logic), but you cannot make changes.

The background color indicates the current compile state of the logic. If the logic source code has not been edited since it was last compiled, the logic has a light green background.

If changes to the source code have been made, but the source has not been compiled, the logic will have a light red background. The resource tree entry will also be displayed in red.

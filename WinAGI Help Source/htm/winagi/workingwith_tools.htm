Working with Tools

In addition to resource editors, WinAGI provides the following tools to make it easier to create AGI games:

Layout Editor

The Layout Editor provides a graphical environment to organize the rooms in your game.

Menu Editor

Use the menu editor to easily create and edit menus.

Text Screen Editor

The Text Screen Editor provides a simple WYSIWYG (what you see is what you get) editor to assist with creating text screen displays in you game (for example, help screens).

Global Defines

Global defines provides an easier way to manage defined names to improve readability of logic source code.

Reserved Defines Editor

The reserved define names that WinAGI uses for reserved values can be modified if desired by using the Reserved Defines Editor on the Tools menu.

Code Snippets Manager

Code Snippets provide you with a very handy method for inserting commonly used large (or small) blocks of text quickly and easily.

Color Palette Editor

If you want to design your game to use a color palette other than the AGI default palette, click the Palette button to show the Modify Color Palette dialog box.

Show/Hide Warning and Errors List

This option toggles the Warning and Errors List between open and closed.

Character Map Selection Tool

To easily add extended characters to resource editors that manipulate text, use the Character Map dialog box.

Custom Tools

WinAGI provides a customizable Tool menu that allows you to launch other programs, link to web pages or link to documents.

Creating a New AGI Game

Creating New Games

You can create a new AGI game from a template, or as a blank set of game files by choosing New Game from the Game menu.

When creating a new game, the Game Properties dialog is displayed to allow you to choose the target directory, GameID, Interpreter version, and other properties. You must choose a valid directory location for your game, and assign a GameID. Other properties can be set now, or can be left to their default values and changed later by reopening the Game Properties dialog.

The chosen directory for a new game must not already contain a WinAGI game file or an existing AGI game (VOL and DIR files).

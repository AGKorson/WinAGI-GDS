Working with Resources

The four main resource types in AGI are the logics, pictures, sounds, and views. The WORDS.TOK file (vocabulary words in a game) and the OBJECT file (list of inventory items) are also considered resources.

Topics in this section provides detailed information about working with resources in WinAGI.

Creating new resources, opening resources for editing, importing and exporting resources, and adding/removing resources to/from games is discussed in this topic.

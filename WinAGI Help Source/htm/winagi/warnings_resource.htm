Game and Resource Warnings

These warnings represent issues identified in required game files and resources when importing or opening an AGI game. Warnings do not prevent WinAGI from opening resources and files, but the issues identified should be corrected.

Detailed explanations of each warning that may be encountered are provided below.

Where resource error messages include dynamic text, such as a resource type, or detailed error information, the dynamic text is represented by '<' and '>' characters, for example '<restype>'.

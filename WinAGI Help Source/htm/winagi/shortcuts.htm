Summary of Keyboard Shortcuts

Here is a list of all keyboard shortcuts available in WinAGI:

Game Shortcuts

F1: Context Sensitive Help

F2: Game Settings

F4: Game Properties

F11: AGI Reference

ALT+X: Close Game

CTRL+B: Compile Game

CTRL+N: New Game from Template

CTRL+O: Open Game

CTRL+I: Import Game

CTRL+R: Run Game

CTRL+SHIFT+B: Compile To...

CTRL+SHIFT+D: Compile Changed Logics

CTRL+SHIFT+R: Rebuild VOL Files

CTRL+SHIFT+N: New Game from blank

ALT+F1: AGI Logic Command Help

ALT+F4: Exit WinAGI

Resource Shortcuts

ALT+1 - 7: Open Resource

CTRL+1 - 7: New Resource

CTRL+ALT+1 - 6: Import Resource

ALT+N: Renumber Resource

CTRL+D: Resource Description & ID

CTRL+E: Export Resource/Save As...

CTRL+F: Search Logics for Selected Resource ID

CTRL+SHIFT+A: Add/Remove Resource to/from Game

CTRL+S: Save Resource

Tools Shortcuts

CTRL+L: Layout Editor

CTRL+M: Menu Editor

CTRL+T: Text Screen Editor

CTRL+G: Global Defines Editor

CTRL+W: Reserved Defines Editor

CTRL+SHIFT+T: Logic Code Snippets Manager

CTRL+SHIFT+P: Palette Editor

CTRL+SHIFT+W: Show/Hide Warning List

F6: Customize Tools Menu

Logic Editor Shortcuts

DEL: Delete Text

F3: Find Again

F8: Compile Logic

ALT+B: Comment Block

ALT+M: Message Cleanup

ALT+U: Uncomment Block

CTRL+A: Select All

CTRL+C: Copy Text

CTRL+F: Find

CTRL+H: Replace

CTRL+SHIFT+J: Insert Defines

CTRL+SHIFT+T: Create/Insert Snippet

CTRL+V: Paste Text From Clipboard

CTRL+X: Cut Text

CTRL+Y: Redo

CTRL+Z: Undo

CTRL+SHIFT+S: Save All Open Logics

CTRL+INS: Show Character Map Tool

TAB: Block Indent

SHIFT+TAB: Block Outdent

DEL: Delete Text

CTRL+Y: Redo

Picture Editor Shortcuts

DOWN_ARROW: Select Next Command/Coordinate

ALT+S: Toggle Visual/Priority Screen

CTRL+V: Paste Commands

CTRL+X: Cut Commands

CTRL+Z: Undo

SHIFT+DEL: Clear Picture

SHIFT+INS: Insert Coordinate

CTRL+SHIFT+J: Join Commands

CTRL+SHIFT+T: Split Command

CTRL+SHIFT+S: Save Image As...

CTRL+ALT+B: Load/Configure Background

CTRL+ALT+C: Copy Commands as Text

CTRL+ALT+P: Adjust Priority Band Base

SHIFT+ALT+B: Remove Background

View Test Mode:

ALT+V: Switch to View Test Mode

NUM PAD KEYS and ARROW KEYS: Control View Object Motion

SPACE: Toggle Status Bar Display

ALT+O: Set Test Options

ALT+SHIFT+V: Choose a Test View

Text Test Mode:

ALT+T: Switch to Text Test Mode

ALT+D: Text Test Mode Options

Sound Editor Shortcuts

BACKSPACE or DEL: Delete Notes

LEFT_ARROW: Move Cursor Left

RIGHT_ARROW: Move Cursor Right

SHIFT+LEFT_ARROW: Adjust Selection Left

SHIFT+RIGHT_ARROW: Adjust Selection Right

ALT+D: Shift Notes Down

ALT+U: Shift Notes Up

CTRL+A: Select All Notes In Track

CTRL+C: Copy Notes

CTRL+K: Toggle Keyboard

CTRL+V: Paste Notes

CTRL+X: Cut Notes

CTRL+Z: Undo

View Editor Shortcuts

DEL: Delete Loop/Cel

CTRL+A: Select Entire Cel

CTRL+C: Copy Loop/Cel/Area

CTRL+V: Paste Loop/Cel/Area

CTRL+X: Cut Loop/Cel/Area

CTRL+Z: Undo

CTRL+INS: Show Character Map Tool (only when editing ViewDescription property)

SHIFT+INS: Insert Loop/Cel

SHIFT+DEL: Clear View/Loop/Cel

SHIFT+T: Toggle Preview Transparency

UP_ARROW/DOWN_ARROW/LEFT_ARROW/RIGHT_ARROW: Move Up in /

'-': Decrease Editor Scale

CTRL+'=': Increase Preview Scale

CTRL+'-': Decrease Preview Scale

SPACEBAR: Start/Stop Preview Cycling

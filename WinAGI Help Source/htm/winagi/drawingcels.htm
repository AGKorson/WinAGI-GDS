Drawing View Cels

The drawing features available in WinAGI for drawing cels are basically the same found in most simple paint programs, such as Microsoft Paint.

In addition to standard drawing tools, you can resize cels, set the transparent color and flip cels and selected areas.

The status bar displays useful information while drawing cels.

 

Drawing Tools

None

If you don't want an active tool, click the None toolbar button. Mouse clicks on the drawing surface are ignored when the None tool is selected.

Area Select

Use the Area Select tool to select rectangular areas of the cel. Once selected, you can cut, copy or delete the selection. Use the flip commands to flip the selection.

You can also drag the selection to a different location in the cel.

The selection normally contains all the pixel data as selected, including pixels that match the cel's transparent color. If you want the transparent color pixels to not overwrite existing pixels when you move or paste a selection, enable 'Transparent Selection' on the Edit menu. When selected, all pixels in the selection that match the cel's transparent color will not overwrite the underlying cel pixels.

Pencil

The Pencil tool is used to draw pixels. The color used depends on which mouse button is used. Use the color palette to change the colors assigned to each mouse button.

Line

Straight lines can be drawn with the Line tool. Press down either mouse button (depending on desired color) to start a line. Drag the cursor, and release at the desired end point to finish the line.

Rectangle

Open rectangles can be drawn using the Rectangle tool. Press down either mouse button (depending on desired color) to set the starting corner, then drag and release at desired end point to finish the rectangle.

Filled Rectangle

The filled rectangle is similar to the Rectangle tool. The difference is the filled rectangle creates a solid rectangle of the chosen color.

Fill

Use the Fill tool to flood fill an area. The fill will extend in all directions until a different color or a boundary is reached.

Erase

The Erase tool is similar to the Pencil tool, except both buttons will reset cel pixels to the current transparent color for this cel.

 

Resizing Cels

To resize cels, place the mouse cursor on the right or bottom edge of the cel. The cursor will change to allow resizing. Left click and drag the border to change the size. You can adjust height and width at same time by dragging the lower right corner.

You can also resize cels by adjusting the Width and Height properties in the Properties Window. Double-click the desired property and enter the desired value. WinAGI will limit the input to a minimum of 1 and a maximum of 160 (width) or 168 (height).

Transparent Color

The transparent color is the color that will be 'masked out' when the cel is drawn on the screen in an AGI game. To change the transparent color, double click on the TransCol property. Choose the desired value from the drop down list.

You can also press CTRL and left click a color on the palette to change the transparency color.

Flipping

Use the flip toolbar buttons to flip cels or selected cel areas horizontally or vertically.

If there is an active selection (the selection boundaries are visible and flashing), the flip tools flip the selection only.

If there is no active selection, the entire cel will be flipped.

Status Bar

When drawing cels, the status bar shows the current image scale, the current tool, and the mouse cursor coordinates.

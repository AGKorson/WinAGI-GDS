Sound Editor

With WinAGI's Sound Editor, you can easily create and modify AGI sound resources. Currently only PC/PCJr sounds are editable in WinAGI. Apple IIgs resources (PCM and MIDI) are currently not editable, but they can be previewed.

See the Creating & Editing Sounds topic for help with using sound editor commands to create and edit sounds.

See the Music Primer topic for a basic overview of the music staff notation used in the WinAGI Sound Editor.

 

Sound Editor Components

For a description of the Sound Editor window components, click on the area of interest:

Toolbar

The toolbar provides easy access to several editing functions, including staff scale control, note shifting, volume control, and cut, copy, paste, and delete commands. Play and stop buttons allow you to easily preview your sound during editing.

Note List

The note list is a hierarchical list of the three music tracks, the noise track and the notes in each track. The item selected in the note list determines what editing commands are available, which track is selected, and the position in the track where editing will occur.

The note list is synchronized with the music staves. Selecting a note in the note list will automatically scroll that note into view on its staff (if the staff is visible).

Music Staves

(Plural of 'staff' is 'staves'.)

Each music staff provide a graphical display of a track's notes. Use the Visible property to show or hide the staff for any of the four available sound tracks.

You can also have the editor display only the currently selected track (temporarily hiding all the others) by setting the "Show Only Selected Track" option on the Resource menu.

The currently selected track will be outlined in blue. The current note position will be marked by a flashing vertical cursor. If one or more note is selected, the selected notes will be highlighted.

If a note's length corresponds to a standard MIDI length (an eighth note, a quarter note, a half note, etc.), it will display the appropriate note symbol. If not, the note is displayed as a bar where the bar's length equals the note's length. You can force the editor to display all notes as bars by clearing the 'Use Note Bitmaps for Standard Note Lengths' option on the Sound tab of the Settings form.

Each track includes time markers that correspond to the length (in seconds) of a whole note based on the sound's Ticks Per Quarter Note value. Keep in mind that the AGI sound format specifies the length of each note in 1/60th second units. In order to be able to display sounds as standard notes, the TPQN value must be a multiple of 4. This is why the time markers will never align with whole seconds.; for example, a whole note that is 64 ticks long will be 64/60 = 1.07 seconds.

Each track also includes a mute button in the upper left corner. You can easily change a track's mute property by clicking on the track's mute button. The mute property only affects playback of the sound while it is being edited.

Scale Control

  You can adjust the display scale by using the scale in/scale out toolbar buttons. The minimum scale is one, maximum is three.

Keyboard

A virtual piano-style keyboard is available to enter notes. Click a key to add the note to the currently selected track at the current cursor position.

The cursor must be active to insert a note. If the selected item in the note list is the sound node (top item in the list) or one of the track nodes, the keyboard will sound the note when clicked, but nothing will be inserted.

To access keys that are not visible, use the scroll buttons to the left of the keyboard. The numbers indicate the octave of each set of notes. Octaves increase going from left to right.

When the noise track is selected, the keyboard displays a custom keyboard that allows entry of noise notes.

You can hide or show the keyboard and note control sections by selecting the Hide Keyboard and Show Keyboard commands on the Edit menu. You can also control the default value for the keyboard section on the Sounds tab of the Settings window.

You can also enter noted directly from the keyboard by typing the desired letter ('g' through 'f'). The note played will be from the currently selected octave. To change the current octave, use the "<" and ">" keys.

Note Control

In the note control section, you can set the properties of the notes being entered. This section DOES NOT modify an existing note; it affects notes that you add via the virtual or real keyboard.

The scroll bar buttons to the right of the note control area allow you to set the length of new notes. The range is from a sixteenth note up to a whole note.

The plus and minus signs above the note allow you to adjust the attenuation of the note. The color of the note indicates the attenuation. The colors match the standard AGI EGA color palette. Full volume (attenuation of 0) is black.

You can toggle between playing a note and playing a rest (silence)by using the space bar.

Properties

The property window displays a list of properties corresponding to the currently selected sound component. The following tables describe the properties available for each:

When the Sound Resource node is selected:

Property Description Read Only?
ID The Resource ID for the sound. Y
Description The resource description for the sound. Y
Key The music key used when displaying notes on the music staff. (This is a WinAGI specific property and has no effect on the actual sound resource.) N
TPQN Ticks per Quarter Note- This is the number of AGI ticks that make up a quarter note as displayed on the staff. Adjusting this value does not affect the Sound resource. TPQN must be a multiple of 4, within the range of 4 to 64. (This is a WinAGI specific property and has no effect on the actual sound resource.) N
Length The length of the entire sound resource, in seconds. Equal to the length in ticks divided by 60. Y

 

When music track node is selected:

Property Description Read Only?
Instrument The instrument that WinAGI will use when playing the sound as a MIDI file. It has no effect when the playback mode is PCJr emulation. (This is a WinAGI specific property and has no effect on the actual sound resource.) N
Muted If TRUE, this track will not be included when the sound is previewed. (This is a WinAGI specific property and has no effect on the actual sound resource.) N
Visible If TRUE, the staff for this track will be visible. A staff does not have to be visible in order to add and edit notes. This property is temporarily disabled when the Show Only Selected Track option is active. (This is a WinAGI specific property and has no effect on the actual sound resource.) N
Note Count The total number of notes in this track. Y

 

When the noise track node is selected:

Property Description Read Only?
Muted If TRUE, this track will not be included when the sound is previewed. (This is a WinAGI specific property and has no effect on the actual sound resource.) N
Visible If TRUE, the staff for this track will be visible. Staves do not have to be visible in order to add and edit notes. This property is temporarily disabled when the Show Only Selected Track option is active. (This is a WinAGI specific property and has no effect on the actual sound resource.) N
Note Count The total number of notes in this track. Y

 

When a music note node is selected:

Property Description Read Only?
Freq Div The frequency divisor is a 10 bit number (0 - 1023) used to calculate the frequency of a tone played by the AGI interpreter. N
Duration The duration of the note (in 1/60th second increments) N
Attenuation Attenuation determines the loudness of the note. No attenuation (0) means full volume. Full attenuation (15) means the note is silent. N
MIDI Note This is the equivalent music note that corresponds to the frequency divisor value. See the music primer for more information. N
MIDI Octave The octave of the MIDI note. See the music primer for more information. N
MIDI Length The duration of the note as a standard MIDI length (eighth note, quarter note, etc.) If the actual duration does not correspond to a standard MIDI length, this value will be undefined. N

 

When a noise note node is selected:

Property Description Read Only?
Type Noise notes can be either periodic tone or white noise. N
Freq Div Noise notes can be one of three frequencies, or can borrow the frequency from track 3. See the AGI Sound resource topic for additional information. N
Duration The duration of the note (in 1/60th second increments) N
Attenuation Attenuation determines the loudness of the note. No attenuation (0) means full volume. Full attenuation (15) means the note is silent. N

WinAGI-specific Properties

The Key, TPQN, Instrument, Muted and Visible properties are WinAGI-specific - they have no affect on the sound resource that is stored in a game's VOL files. They are provided only to help with the creation and editing of sounds.

Resource Load Error RE08

Logic <number> source file (<filename>) is invalid due to file access error

This error is raised if WinAGI is unable to access the source file of the specified logic. It may be due to a disk error or the file may be locked by another process. It may also be a corrupt file.

If the problem is caused by a disk error or locked file, close the source file and address the issue outside of WinAGI. Then try reopening the source file.

If the problem is a corrupt file, a backup or archived version of the file is the only way to recover the source file.

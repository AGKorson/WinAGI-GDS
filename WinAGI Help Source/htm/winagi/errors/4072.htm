Logic Compiler Error 4072

Message <number> is already assigned

The compiler adds all messages explicitly assigned with the #messages preprocessor before beginning compilation. When adding preassigned messages, if you attempt to assign more than one message to the same number, this compiler error is raised.

#message 1 "Hello world."
#message 1 "Good bye world." [ NOT OK; message 1 already assigned

Logic Compiler Error 6012

Invalid name in <preprocessortype> preprocessor statement

Only applicable when using Sierra syntax.

The name portion of a preprocessor statement cannot be a number or symbol or previously defined name. It must be a unique identifier.

%test ==(VAR, NUM) 1  [ NOT OK; name can't be a symbol
%define cmd1   1
%action cmd1(VAR, NUM) 1  [ NOT OK; duplicate name
...

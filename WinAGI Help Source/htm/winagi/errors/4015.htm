Logic Compiler Error 4015

'<label>' is a <tokentype>, and cannot be used as a label

Tokens that are already in use for other purposes cannot be used as a label name:

    • numbers
    • global defines or local defines
    • argument markers
    • keywords
    • action commands
    • test commands
    • reserved numerical constants (edge codes, computer types, sound types)
    • reserved flags
    • reserved message constants (gameVersionMsg, gameAboutMsg, gameID)
    • reserved object or string name (ego) or (input_prompt)
    • reserved variables

This compiler error is raised if you attempt to use any of the above as a label name.

...
goto(if);  [ NOT OK
...
:if  [ NOT OK
...

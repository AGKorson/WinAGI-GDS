Logic Compiler Error 4007

'}' expected

This error occurs when the end of the file is reached and one or more if and/or else blocks do not have a closing curly brace. This is often caused by cutting and pasting errors.

Using indentation in your source code can help spot this error.

if (v99 == 1) {
    v100 = 1;
return(); [ end-of-file  NOT OK missing '}'

Logic Compiler Error 4046

Defines <define1> and <define2> make a circular reference

Defines are allowed to redefine other defines. For example:

#define avar   v99
#define bvar   avar  [ OK, same as #define bvar v99
...

This error is raised if the compiler detects defines with values that refer to each other such as: 

#define avar   bvar
#define bvar   avar  [ NOT OK, circular reference
...
#define cvar   dvar
#define dvar   evar
#define evar   cvar  [ also NOT OK, circular reference
...

Resource Load Error RE12

View <number> has no loops

AGI view resources must have at least one loop, and each loop must have at least one cel. This error is raised when a view has no loops. This is most likely a corrupt view.

WinAGI will treat the view as having a single loop with one default cel as a placeholder. Opening the view in WinAGI and then saving it will clear this error.

Logic Compiler Error 4018

'==' is a comparison operator and cannot be used for variable, flag or string assignment

When using alternate syntax to assign variables, flags or strings, use a single equals sign '='. The double equals sign '==' is only valid in comparison statements.

v99 == 99; [ NOT OK
v99 = 99; [ OK
...
if (v99 = 99) [ NOT OK
if (v99 == 99) [ OK
...

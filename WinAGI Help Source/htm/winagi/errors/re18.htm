Resource Load Error RE18

Invalid OBJECT file header, unable to read item data

In order to extract the item text and room data, the OBJECT file should have a properly formatted header. This error indicates WinAGI was unable to validate the header.

The OBJECT file is corrupt and most likely unrecoverable. WinAGI treats the file as having a single null item. You will need to re-enter items manually to recover.

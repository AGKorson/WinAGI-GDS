Logic Compiler Error 4023

Unexpected symbol (<definevalue>) or syntax error in assignment statement

This compiler error indicates an invalid symbol or operator was encountered instead of a command or end of line marker. This can be caused by extra spaces when trying to do indirect variable assignments or other typographical errors.

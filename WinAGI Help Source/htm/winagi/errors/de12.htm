Logic Source Decompiler Error DE12

Word group <number> does not exist in WORDS.TOK

Only applicable when using Sierra syntax.

This error is raised when the decompiler finds an invalid word group number while decompiling a said command. Sierra syntax requires the argument to be the word text, but because the word does not exist, the word group number is used as a place holder.

This error may occur if the WORDS.TOK file used to decompile the logic is not the same as the one originally used to create the logic. It could also mean either the logic or the WORDS.TOK file is corrupted. You should examine the logic and WORDS.TOK file carefully and make corrections as needed.

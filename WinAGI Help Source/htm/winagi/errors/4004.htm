Logic Compiler Error 4004

Undefined token '<token_text>' for argument <number>: expected <type>

If the compiler encounters an undefined token, this compiler error is raised. This is most likely caused by typographical errors, but can also occur if the Use Reserved Define Names property is disabled for logics that previously used reserved names. Check the spelling of variable names, re-enable the Use Reserved Define Names option, or add a global or local define name to correct this.

#define bear   o3 
animate.obj(bare);  [ NOT OK; check your spelling!

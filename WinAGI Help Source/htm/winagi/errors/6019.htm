Logic Compiler Error 6019

Right indirection only valid with variable arguments

Only applicable when using Sierra syntax.

When using alternate syntax for the rindirect command, the assigned value (on right side of '=@') must be a declared variable. This error indicates an attempt to use right indirection with a number argument.

v99 =@ 1;    [ NOT OK; second argument must be a variable
v99 =@ v1;   [ OK

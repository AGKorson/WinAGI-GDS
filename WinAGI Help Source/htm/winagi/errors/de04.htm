Logic Source Decompiler Error DE04

Block end at resource index <pos> is past end of resource; adjusted to end

This error is raised when the calculated end of an if block is past the code section of the logic. If the if condition is FALSE, AGI would jump to an invalid location past the end of the valid logic code, into the message section or beyond.

This is likely due to a corrupt logic resource, or an error that existed in the original Sierra release disks. If it is an original error, it probably is in abandoned code that never gets run, otherwise the error would have been quickly identified.

WinAGI sets the block end to the end of the resource and will try to continue decompiling. Regardless, you should confirm that the VOL file containing this logic is not corrupt. Regardless, you will need to carefully examine the decompiled source code, and adjust the code as necessary before recompiling.

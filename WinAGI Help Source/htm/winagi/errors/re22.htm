Resource Load Error RE22

WORDS.TOK file contains no data

The minimum size of a valid WORDS.TOK file is 52 bytes (assuming no words, which would work in AGI, but is unlikely to be of usefulness) or 56 bytes if only a single one-word entry is in the file. This error indicates the file is too small to be valid.

WinAGI treats the file as an empty word list. Opening and then saving the file in WinAGI will clear the error.

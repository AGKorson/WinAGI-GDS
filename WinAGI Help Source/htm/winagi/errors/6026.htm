Logic Compiler Error 6026

'<token>' is a key word or default compiler define name

Only applicable when using Sierra syntax.

Compiler key words and default define names cannot be used as the name for another define declaration. Key words and default defines include:

    • if
    • else
    • goto
    • %include/#include
    • %tokens/#tokens
    • %action/#action
    • %test/#test
    • %flag/#flag
    • %var/#var
    • %object/#object
    • %view/#view
    • %define/#define
    • %message/#message
    • NUM
    • FLAG
    • VAR
    • OBJECT
    • VIEW
    • MSGNUM
    • WORDLIST
    • ANY
    • MSG
    • WORD

Check the define name for possible misspelling (remember names are case sensitive) and adjust as necessary to clear this error.

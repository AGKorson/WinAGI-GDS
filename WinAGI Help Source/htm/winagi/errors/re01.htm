Resource Load Error RE01

<restype> <number> is in a VOL file (<volfile>) that does not exist

If the VOL file value from the DIR file for this resource is for a non-existent VOL file, this error is raised. It usually indicates an abandoned resource where the DIR file was never updated. There are several instances of this error in Sierra games. Deleting the resource is the only way to fix this anomaly.

It may also indicate a VOL file was accidentally deleted from your game directory.

If the file was accidentally deleted, you will need to close the game, restore the missing VOL file, then re-open the game to clear the error.

If the resource resource is not valid, remove the resource from the game to clear this error.

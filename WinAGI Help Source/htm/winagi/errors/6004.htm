Logic Compiler Error 6004

'<definename>' is already defined; duplicate define names are not allowed

Only applicable when using Sierra syntax.

To maintain compatibility with Sierra's logic compiler CG.EXE, define names (those set by %object, %variable, %flag, %view and %define) must be unique. You will need to adjust the affected define names as needed to make them all unique.

%var v99   99
...
%define v99   99   [ NOT OK
...

Logic Compiler Error 4025

Incorrect type or syntax error in unary (++/--) statement

WinAGI accepts the alternate syntax '++v0' and '--v0' as substitutes for the increment and decrement commands. This error is raised if the compiler encounters a '++' or '--' entry without a corresponding variable. This is often due to a spelling error or failure to properly define a variable.

#define somevar   v99
soemvar++;  [ NOT OK; misspelled 'somevar'
...

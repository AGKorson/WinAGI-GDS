Logic Compiler Error 6001

Include file stack overflow; limit is five

Only applicable when using Sierra syntax.

This error indicates the nesting level of include files is too large. Nesting means the number of times one included file references another include file - for example if a logic includes "file1.txt", and file1.txt includes "file2.txt", the nesting level of file2.txt is two. To maintain compatibility with Sierra's compiler CG.EXE, the nesting level limit is five.

To correct this error you will need to reconfigure your include files so the nesting level is five or less.

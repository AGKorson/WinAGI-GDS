Logic Compiler Error 4036

Invalid preprocessor symbol '<symbol>'

The only valid preprocessor symbols are #include, #define and #message. This error indicates a token was encountered beginning with a '#' that is not a valid token. Tokens are case sensitive.

This error is also raised if '%' character is used with normal syntax preprocessor tokens. Although allowed in original Sierra syntax, valid preprocessor symbols in AGI fan syntax can only start with '#'.

#defien var2   v100  [ NOT OK; misspelled 'define'
#INCLUDE "file1.txt"   [ NOT OK; not all lower case
%message 1 "Hello world."   [ NOT OK in Fan syntax
#define var1    v99  [ OK
...

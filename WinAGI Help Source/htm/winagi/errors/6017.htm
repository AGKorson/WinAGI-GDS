Logic Compiler Error 6017

Invalid action command '<token>'

Only applicable when using Sierra syntax.

This error indicates the compiler was expecting an action command or variable as part of an alternate syntax statement, but an unexpected token such as a number was found. This is usually caused by a syntax error in source code. Check you code carefully, and correct as needed.

Logic Compiler Error 6020

WORDS.TOK file not loaded with %tokens preprocessor

Only applicable when using Sierra syntax.

To maintain compatibility with Sierra's logic compiler CG.EXE, if you attempt to use the said command but haven't used the %tokens preprocessor to load the game's WORDS.TOK file, this error is raised.

To clear this error, you will need to add the %tokens preprocessor to your source code, or remove the said command.

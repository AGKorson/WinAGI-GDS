Logic Compiler Error 4060

'<symbol>' is not a valid comparison operator

Allowable comparison operators for AGI are '==', '!=', '<', '>', '<=' and '>='. This error is raised if a non-allowable symbol is detected.

The most likely cause is a typo in your code. Using a single equal sign instead of the double-equal, or putting the equal sign before the less-than or greater-than are common errors that you will need to correct if necessary.

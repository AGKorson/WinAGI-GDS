Logic Compiler Error 4052

Invalid string value; unable to concatenate

When concatenating text string values to create a larger string, each segment needs to be a text field surrounded by quote marks, with only spaces or new line characters separating them. If any of the segments do not have an ending quote mark, this error is raised.

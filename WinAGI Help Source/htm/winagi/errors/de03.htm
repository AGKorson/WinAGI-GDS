Logic Source Decompiler Error DE03

Invalid action command byte code (<code>) found at resource index <pos>

This error is raised when the decompiler is expecting a valid action command or key word control code, but instead encounters a byte code value that is neither. It is not possible to continue decompiling when this error occurs.

This is likely due to a corrupt logic resource, VOL or DIR file. Dependng on when the error occurred, there may be a portion of the logic source recovered, but regardless, you will need to adjust the source code will need

WinAGI sets the block end to the end of the resource and will try to continue decompiling. Regardless, you should confirm that the file containing this logic is not corrupt. Regardless, you will need to carefully examine the decompiled source code, and adjust the code as necessary before recompiling.

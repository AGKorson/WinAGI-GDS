Logic Compiler Error 4054

Nothing allowed on line after preprocessor declaration

Preprocessor commands (#include, #define, #message) must be declared on a single line by themselves. Comment are allowed, but nothing else.

#include "file1.txt"   [ OK - one preprocessor command, and a comment
...
#include "file1.txt" #include "file2.txt"  [ NOT OK; each
                      [ preprocessor command must be on its own line
...
#define v220 tempvar1; [ NOT OK, don't use semicolon line-end marker
...
#define v221 tempvar2 = 1;   [ NOT OK, can't combine define and commands
...
#define v224 tempvar5  [ OK, single preprocessor with nothing after
...

Logic Compiler Error 6007

Number expected in <preprocessortype> preprocessor statement,'<token>' is not a number

Only applicable when using Sierra syntax.

To properly define an action or test command, you must include the byte value to be assigned to the command immediately after the argument list. This error indicates something other than a number (or a define name that has a number value) was found after the argument list.

You will need to edit the command declaration to include the correct byte value.

%action assignn(VAR, NUM)   [ NOT OK; missing number
%action assignv(VAR, VAR) v2  [ NOT OK; not a number
%action addn(VAR, NUM) 3  [ OK
...

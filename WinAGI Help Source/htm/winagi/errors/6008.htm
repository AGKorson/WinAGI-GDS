Logic Compiler Error 6008

Invalid command number in <commandtype> command preprocessor statement

Only applicable when using Sierra syntax.

This error indicates the byte value assigned to the action or test command is not a valid AGI command number.

The maximum AGI action command value depends on the target interpreter version. Attempting to define a command that is not valid for the game's target version will raise this error.

AGI test command values are limited to 1 - 18 for all AGI versions.

To correct this error, you need to ensure you have the correct command number. If the error is due to a command not supported for the game's target interpreter version, you can use the Game Properties dialog to adjust the version to one that does support the command in question.

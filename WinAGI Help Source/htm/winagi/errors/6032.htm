Logic Compiler Error 6032

'if' block must have at least one test command

Only applicable when using Sierra syntax.

Although AGI allows if blocks with no test commands when processing a logic at run time, Sierra's logic compiler CG.EXE will raise an error when it detects an empty if block.

To maintain compatibility with CG.EXE, WinAGI also raises this error for empty if blocks. You will need to edit your code to add test commands, or remove the empty blocks.

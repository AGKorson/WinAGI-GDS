Logic Compiler Error 4089

<filename> is a required #include file for this logic

When using WinAGI's auto-include options for reserved items, global defines, and resource IDs, WinAGI automatically adds the appropriate #include preprocessor to each source file. If any of these #include lines are missing, this error is raised.

To fix this error, either add back the missing #include statement, or re-save the source in WinAGI, which should automatically reinsert the line.

Logic Source Decompiler Error DE02

Invalid test command byte code (<code>) found at resource index <pos>

This error is raised when the decompiler is expecting a valid test command or if block control code, but instead encounters a byte code value that is neither. It is not possible to continue decompiling when this error occurs.

This is likely due to a corrupt logic resource or corrupt VOL or DIR file.

WinAGI sets the block end to the end of the resource and will try to continue decompiling. Depending on where the error occurred, there may be a portion of the logic source recovered, but regardless, you will need to examine and manually repair the logic source code.

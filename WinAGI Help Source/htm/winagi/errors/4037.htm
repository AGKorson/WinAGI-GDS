Logic Compiler Error 4037

Incorrect type for argument <number>: '<define_name>' is not a <type>

If the compiler detects a define name of the wrong argument type, this compiler error is raised. This usually means the wrong argument was used by accident, or perhaps a typographical error. Replace the argument with a define name or argument marker of the correct type to correct this error.

#define aflag   f99
#define avariable   v99
assignn(aflag, 0);  [ NOT OK; expecting a variable
assignn(avariable, 0);  [ OK
...

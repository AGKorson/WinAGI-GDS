Logic Compiler Error 4019

AND operator (&&) not allowed within brackets

If test commands are AND'ed, they cannot be enclosed in brackets. (OR'ed commands must be enclosed in brackets.)

This compiler error occurs if the compiler detects AND'ed tests ('&&') that have been mistakenly placed in parentheses.

if(  (v99==99 && v100==100)  ) [ NOT OK
...
if(v99==99 && v100==100) [ OK
...

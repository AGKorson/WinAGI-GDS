Logic Source Decompiler Error DE08

Message <msgnum> has an invalid offset (resource index <pos>)

AGI uses a table of offset values to identify the starting location of each message in the message data section of a compiled logic.

This warning indicates the message offset value points to a location outside the message data section. It most likely means the logic data is corrupt. WinAGI will mark the message as unused since there is no valid data to assign it. You should examine the decompiled source code closely, make any corrections needed, then recompile to fix this issue.

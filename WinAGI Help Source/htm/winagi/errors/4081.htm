Logic Compiler Error 4081

Syntax error in indirection assignment

There are only three forms allowed for using alternate syntax to represent indirect variable assignments. This error indicates a form other than the allowed was detected by the compiler.

*v99 = 100;    [ OK, left numeric indirection assignment
*v99 = v100;   [ OK, left variable indirection assignment
v99 = *v100;   [ OK, right variable indirection assignment
v99 = *v99 + 1;      [ NOT OK
v99 = v99 + *v100;   [ NOT OK
*v99 = *v100;        [ NOT OK

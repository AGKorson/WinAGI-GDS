Logic Compiler Error 6010

Invalid argument type in <commandtype> command preprocessor statement

Only applicable when using Sierra syntax.

When assigning the argument list to commands, the types MSG, WORD and ANY are not valid for any command. Also, the type WORDLIST is only valid for the said test command (command number 14). You will need to change the type to a supported value: NUM, FLAG, VAR, OBJECT, MSGNUM or VIEW.

%action print(MSG) 101  [ NOT OK; MSG is not a usable argument type
%test said(WORD) 14  [ NOT OK; WORD is no a usable argument type
...

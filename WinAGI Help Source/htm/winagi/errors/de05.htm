Logic Source Decompiler Error DE05

Destination for 'goto' at resource index <pos> is past end of logic; adjusted to end

This warning is similar to warning DC08, with the difference being that the problem is associated with a goto byte code instead of being in an if-block. It is raised when a goto would jump to an invalid location past the end of the valid logic code, into the message section or beyond.

This is likely due to a corrupt logic resource, or an error that existed in the original Sierra release disks. If it is an original error, it probably is in abandoned code that never gets run, otherwise the error would have been quickly identified.

You should confirm that the VOL file containing this logic is not corrupt. Regardless, you will need to carefully examine the decompiled source code, and adjust the code as necessary before recompiling.

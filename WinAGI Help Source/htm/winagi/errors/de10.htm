Logic Source Decompiler Error DE10

Unexpected end of input

This error indicates the decompiler was expecting more data in the logic resource (for example, when parsing a command, it expects all the command's arguments) but the end of the file was reached. If this happens, it most likely means a serious corruption error in the file. If WinAGI was able to extract any useful data, that will be put in the source file, but it will be incomplete.

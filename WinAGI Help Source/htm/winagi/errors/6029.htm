Logic Compiler Error 6029

Too many levels of nested code blocks

Only applicable when using Sierra syntax.

To maintain compatibility with Sierra's logic compiler CG.EXE, the number of nested code blocks (the number of 'if-else' blocks inside other 'if-else' blocks) is limited to ten.

if (v99 != 1) {
    [ nest level 1
    if (v99 != 2) {
        [ nest level 2
        if (v99 != 3) {
            [ nest level 3
            ...
            [ do not exceed nest level 10
        }
    }
}

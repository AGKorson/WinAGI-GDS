Logic Compiler Error 4009

'}' required before 'else'

The command block immediately preceding the else statement needs to have a closing curly brace. This is often caused by cutting and pasting errors.

Using indentation in your source code can help spot this error.

if (v99 == 1)
    {
    v100 = 1;
else [ NOT OK; missing '}'
    {
    v100 = 2;
    }
...

Logic Source Decompiler Error DE07

Message section has an invalid end-of-text marker

The first two bytes of the message address table in a compiled AGI logic are a pointer to the end of the message section. AGI needs this value so it can properly decrypt the messages when the logic is loaded. In a properly formatted AGI logic, this value should always correspond with the end of the logic data.

This warning indicates the message end pointer is not the expected value. It most likely means the logic data is corrupt. WinAGI will reset the pointer to the expected value, but the messages may not be decompiled correctly. You should examine the decompiled source code closely, make any corrections needed, then recompile to fix this issue.

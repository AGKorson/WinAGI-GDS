Logic Source Decompiler Error DE11

Valid reference to WORDS.TOK is missing

Only applicable when using Sierra syntax.

To maintain compatibility with Sierra's logic compiler CG.EXE, this error is raised if a said command is encountered but the WORDS.TOK file has not been loaded with a %tokens preprocessor command.

To clear the error, add a %tokens preprocessor command that references this game's WORDS.TOK file to either the logic source code, or your gamedefs file, if you are using one.

Logic Compiler Error 4056

Invalid use of 'else'

This compiler error is raised if an else statement is encountered without a corresponding if statement, or if following an existing else block. 

if (isset(f99)) {
    reset(f99);
}
else {
    set(f99);
}
else {   [ NOT OK, 'else' without matching 'if'
    reset(f99);
}

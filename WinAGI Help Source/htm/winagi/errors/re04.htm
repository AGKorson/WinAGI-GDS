Resource Load Error RE04

<restype> <number> has an invalid location (<location>) in volume file <volfile>

If the pointer from the DIR file for this resource is past the end of the VOL file, this error is raised. It usually indicates an abandoned resource where the DIR file was never updated. There are several instances of this error in Sierra games. Removing the resource from your game is the only way to fix this anomaly.

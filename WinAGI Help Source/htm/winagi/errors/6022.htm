Logic Compiler Error 6022

said command arguments must not be enclosed in quotes

Only applicable when using Sierra syntax.

Sierra syntax requires the arguments in said commands to be the word text, not the word group number, without surrounding quotes and with the '$' character used to represent a space character.

To clear this error, remove the quote marks from the word text and replace any spaces with the '$' character.

if (said(9999)) {   [ NOT OK, must be a word text value
...
if (said("rol")) {  [ NOT OK, word must not include quotes
...
if (said(save game)) [ NOT OK (assumes "save game" is a valid word)
...
if (said(rol)) {    [ OK
...
if (said(save$game)) {   [ OK
...

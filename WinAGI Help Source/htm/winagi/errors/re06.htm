Resource Load Error RE06

Unable to expand <restype> <number> at location <location> in <volfile>

In version 3 games, Sierra used compression to reduce the size of resources in VOL files. This error indicates WinAGI attempted to decompress a version 3 resource, but was unable to output a valid resource.

This error probably indicates a corrupted VOL file. The resource is probably unrecoverable. You can remove it from your game to clear this error.

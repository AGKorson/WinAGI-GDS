Logic Compiler Error 6003

WORDS.TOK filename must be enclosed in quotes

Only applicable when using Sierra syntax.

To maintain compatibility with Sierra's logic compiler CG.EXE, the filename property for the preprocessor command %tokens must be enclosed in double quotes.

%tokens words.tok   [ NOT OK
%tokens "..\\words.tok"  [ OK, includes correct reference to the game's WORDS.TOK file

Logic Compiler Error 4065

Invalid use of 'NOT' operator

In if blocks, the NOT operator (!) must be followed by a test command, or by a variable as a shortcut for the greatern test command. You may not use the NOT operator in front of a variable comparison.

Also, you cannot follow one NOT operator with another, or use the NOT operator in front of an 'OR' bracket.

if (!greatern(v99, 0))  {    [ OK
...
if (!v99) {    [ OK
...
if (!v99 == 0) {    [ NOT OK
...
if (!!v99) {    [ NOT OK
...
if (!(v99 == 0 || v99 == 1)) {    [ NOT OK
...

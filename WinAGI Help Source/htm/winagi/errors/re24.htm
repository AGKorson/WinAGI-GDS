Resource Load Error RE24

globals.txt file not found

If your game is using the IncludeGlobals property, the globals.txt file must be present in your game's resource directory. This flag indicates the globals.txt file is missing, probably from being accidentally deleted or moved.

If the file was accidentally deleted or removed, restore it. You can also clear this error by using the global defines editor to create a new globals.txt.

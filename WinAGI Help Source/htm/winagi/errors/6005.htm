Logic Compiler Error 6005

Argument lists must be enclosed in parentheses

Only applicable when using Sierra syntax.

When defining action and test commands, the argument list must be enclosed in parentheses. If the command has no arguments, an empty set of parentheses are required.

%action addn VAR, NUM  3  [ NOT OK
%action addn(VAR, NUM) 3  [ OK
...
%action return 0  [ NOT OK
%action return() 0  [ OK
...

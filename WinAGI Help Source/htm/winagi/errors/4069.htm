Logic Compiler Error 4069

Include files must be enclosed in double quotes

This error is raised if the filename argument of an #include preprocessor command is not properly delimited by double quotes.

#include textfile.txt     [ NOT OK
#include 'textfile.txt'   [ NOT OK
#include "textfile.txt    [ NOT OK
#include "textfile.txt"   [ OK
...

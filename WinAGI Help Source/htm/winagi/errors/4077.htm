Logic Compiler Error 4077

OR operator or closing bracket expected

This compiler error indicates the compiler is expecting a closing parenthesis or another OR operator to compare commands. All test commands that are OR'ed must be inside a single pair of brackets, and the AND operator can't be included inside the brackets. For example:

if((v99 == 99 || v100 == 100 && v101 == 101)) [ NOT OK
...
if((v99 == 99 || v100 == 100) && v101 == 101) [ OK
...

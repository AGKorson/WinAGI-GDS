Logic Compiler Error 6002

WORDS.TOK filename '<name>' is missing or invalid

Only applicable when using Sierra syntax.

The preprocessor command %tokens requires a valid path and filename to the game's WORDS.TOK file. This error indicates the filename is either missing, or is not a valid filename, or points to a file other than the WORDS.TOK file for the game being edited.

Filenames can use relative paths. In most cases, you should use the name "..\\words.tok" (remembering to use the double-slash so the parser properly recognizes it), assuming source file with the %tokens preprocesser command is located in your game's resource directory.

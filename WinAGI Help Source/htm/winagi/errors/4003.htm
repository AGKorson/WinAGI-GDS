Logic Compiler Error 4003

Complex variable math statements are not allowed

AGI does not support complex math statements such as

v99 = v100 + 1; [ NOT OK 
v99 = v99 + v100 + 2; [ NOT OK
...

Supported math functions include adding/subtracting/multiplying one variable with a number, or another single variable. See the alternate syntax discussion for details on allowed math function syntax.

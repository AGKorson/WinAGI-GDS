Logic Compiler Error 4017

Incorrect type for argument <number>: expected a <type>

If the compiler detects an argument marker of the wrong argument type, this compiler error is raised. This usually means the wrong argument was used by accident, or perhaps a typographical error. To correct this, use an argument marker or a define name of the correct type.

animate.obj(v2);  [ NOT OK; expecting a screen object
animate.obj(o2); [  OK

Logic Compiler Error 6021

Numbers are not valid arguments for the said command

Only applicable when using Sierra syntax.

Although Fan syntax allows the use of word group numbers as valid arguments in said commands, Sierra syntax requires the arguments to be the word text without surrounding quotes and with the '$' character used to represent a space character.

To clear this error, replace the word group number with the text value of one of the group's synonyms.

if (said(9999)) {   [ NOT OK, must be a word text value
...
if (said("rol")) {  [ NOT OK, word must not include quotes
...
if (said(save game)) [ NOT OK (assumes "save game" is a valid word)
...
if (said(rol)) {    [ OK
...
if (said(save$game)) {   [ OK
...

Resource Load Error RE23

Invalid word index table in WORDS.TOK

In a properly formatted WORDS.TOK file there are exactly 26 2-byte index entries in the file header. This error indicates WinAGI was unable to find a valid index in order to extract words.

WinAGI treats the file as an empty word list.

The WORDS.TOK file is corrupt and most likely unrecoverable. You will need to re-enter words manually to recover.

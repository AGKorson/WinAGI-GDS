Logic Compiler Error 4055

Invalid define value '<token>'

Define values can be numbers, literal strings, or standard argument markers (v1, f1, o1, etc.). This error indicates that a value define is none of the acceptable types. It is usually caused by a syntax error.

#define var2   V99 [ NOT OK; upper case not valid for variable marker

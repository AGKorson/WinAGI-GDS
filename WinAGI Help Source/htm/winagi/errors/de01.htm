Logic Source Decompiler Error DE01

Logic contains no data to decompile

The minimum size of a valid AGI logic resource is four bytes - two for the pointer to the message data section, one for a single return command and one for message count of zero.

This error indicates the resource has three or fewer bytes and therefore cannot possibly be a valid logic resource.

You should confirm that the DIR file and VOL file referencing this logic are not corrupt.

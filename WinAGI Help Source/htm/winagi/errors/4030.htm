Logic Compiler Error 4030

'{' must follow an 'if' or 'else' statement

All action commands following an if or else statement must be enclosed in curly brackets '{ }'. This compiler error indicates that the starting curly bracket is missing.

if (v99 == 0)
    v100 = 1;  [ NOT OK; missing '{'
    ...
    }
...

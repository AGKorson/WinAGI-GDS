Logic Compiler Error 4024

Brackets too deep in 'if' statement

You can only have one level of parentheses in if statements to group OR'ed ('||') commands. This error occurs if you try to include extra parentheses inside an OR clause. For example:

if(  (v1==1 || (v2==2 || v3==3))  ) [ NOT OK
...
if(  (v1==1 || v2==2 || v3==3)  ) [  OK
...

Resource Load Error RE02

<restype> <number> is in a VOL file (<volfile>) marked read only

WinAGI requires full read/write access to all game files. This error indicates the specified VOL file is currently set to read only. You will need to change its properties to allow write access.

To clear this error, close the game, adjust the file property to allow full access, then reopen the game.

Logic Compiler Error 4034

Syntax error in string assignment

This compiler error is raised when WinAGI encounters a argument of the argument type that is not a message, a keyword, a command name, or symbol in a string assignment. This is most likely caused by typographical errors. Check your source code and adjust it to include the correct argument type (message).

s1 = v99;  [ NOT OK
s1 = 99;  [ NOT OK
s1 = "v99"; [ OK
s1 = "99"; [ OK
...

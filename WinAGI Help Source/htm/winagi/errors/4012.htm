Logic Compiler Error 4012

Label '<labelname>' not found

This error is raised if the label argument of a goto statement has not been declared in the logic. Most often this means that either the argument value, or the label has a typo in it. To clear the error, edit the argument of the indicated goto statement to a valid label token.

...
goto(Lable1);  [ NOT OK; misspelled 'Label1'
...
Label1:
...

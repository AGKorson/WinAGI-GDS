Logic Compiler Error 4084

')' expected after command arguments

The compiler did not find a closing parenthesis. All commands must include opening and closing parentheses '( )' around argument values. The parentheses are required even if the command doesn't have any arguments.

addn(v99, 1;   [ NOT OK; missing ')'
return;   [ NOT OK; missing '()'
...

This error also occurs when including too many arguments in a command:

addn(v99, 1, 1);   [ NOT OK; addn only takes 2 arguments
...

To clear the error, check your code and confirm the correct number of arguments are included and the ending ')' is included.

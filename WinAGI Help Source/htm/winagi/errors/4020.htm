Logic Compiler Error 4020

AND operator (&&) or end of 'if' statement expected

This compiler error indicates the compiler is expecting the if statement to end (a closing parenthesis) or an AND operator ('&&') to compare commands. For example

if(v99 == 99 && v100 == 100  v101 == 101) [ NOT OK
...

The third test command (v101 == 101) isn't separated by an AND operator.

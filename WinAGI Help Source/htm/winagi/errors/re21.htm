Resource Load Error RE21

File access error, unable to read WORDS.TOK file (<errortext>)

This error is raised if WinAGI is unable to access the WORDS.TOK file. It may be due to a disk error or the file may be locked by another process. It may also be a corrupt file.

If the problem is caused by a disk error or locked file, close the game and address the issue outside of WinAGI. Then reopen the game.

If the problem is a corrupt file, unless you have an backup or archived version of the file to recover, the only remedy is to delete the file and create a new WORDS.TOK file and re-enter words manually to recover.

Logic Compiler Error 4008

'}' not at end of any command blocks

Curly braces are used to designate blocks of code to be executed as part of an if statement or else statement. They cannot be used anywhere else. This is often caused by cutting and pasting errors.

Using indentation in your source code can help spot this error.

 

if (v99 == 1)
    {
    v100 = 2;
  }
}  [ NOT OK

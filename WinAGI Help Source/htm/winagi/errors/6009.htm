Logic Compiler Error 6009

Invalid argument list in 'said' command preprocessor statement

Only applicable when using Sierra syntax.

When defining the said command (byte value of 14), the only allowed argument is WORDLIST. Attempting to define the said command with any other command list will raise this error.

%test said(NUM) 14  [ NOT OK; argument must be 'WORDLIST'
%test said(WORDLIST) 14  [ OK

Resource Load Error RE03

<restype> <number> is invalid due to file access error (<volfile>)

This error is raised if WinAGI is unable to access a VOL file to read the resource. It may be due to a disk error or the file may be locked by another process. It may also be a corrupt file.

If the problem is caused by a disk error or locked file, close the game and address the issue outside of WinAGI. Then reopen the game.

If the problem is a corrupt file, unless you have an backup or archived version of the file to recover, the resource is gone, and you will need to remove it (and likely all other resources from the file) to be able to continue.

Logic Compiler Error 4076

OR operator must be within brackets

When tests are OR'ed ('||'), they must be enclosed in parentheses. For example,

if(  (v98 == 98 || v99 == 99)  ) [ OK
...
if(  v98 == 98 || v99 == 99  )    [ NOT OK
...

This compiler error indicates OR'ed test commands have not been enclosed in brackets.

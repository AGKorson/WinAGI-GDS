Logic Compiler Error 4073

Invalid negative argument value

WinAGI will automatically convert negative numbers into their corresponding unsigned 2's complement value. For example, '-1' is converted to '255', '-2' is converted to '254', and so on. Because all values in AGI are single bytes, negative values lower than '-128' are not allowed.

Negative values are only of practical use when using the reposition command. It is the only command in AGI that uses negative values.

v99 = -128; [ OK
 v99 = -129; [ NOT OK; invalid negative value

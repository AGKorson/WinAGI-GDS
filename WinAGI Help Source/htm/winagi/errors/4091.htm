Logic Compiler Error 4091

Action command <commandname>' not valid in 'if' statement

This compiler error is raised when the compiler expects a test command but an action command was encountered. Action commands are not valid inside an if statement. You will need to replace the action command with an appropriate test command.

if (addn(v99, 1))  [ NOT OK; addn is an action command

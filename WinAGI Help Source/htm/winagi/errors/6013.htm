Logic Compiler Error 6013

Message text is too long

Only applicable when using Sierra syntax.

To maintain compatibility with Sierra's logic compiler CG.EXE, the maximum length of a message is 998 characters.

This error most often occurs when a message string is missing its ending quote. The compiler will assume all the following source code text is part of the string. Adding the missing quote will fix the problem.

If the error is not due to a missing end quote you will need to shorten this message to clear the error.

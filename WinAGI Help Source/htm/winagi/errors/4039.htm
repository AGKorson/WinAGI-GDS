Logic Compiler Error 4039

Invalid argument value for argument <number> (must be between 0-255)

All AGI argument types (except vocabulary words) are limited to byte (0-255) values. This compiler error is raised if you attempt to use a value outside this limit, for example, 'v300' or 'f999'.

addn(v999, 1);  [ NOT OK
if (isset(f300))  [ NOT OK

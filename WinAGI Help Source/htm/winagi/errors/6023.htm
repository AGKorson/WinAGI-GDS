Logic Compiler Error 6023

Undefined message (<messagenum>)

Only applicable when using Sierra syntax.

This error occurs if you attempt to use a message number in an action command, but haven't first declared the message using the %message preprocessor.

To clear this error add a message declaration for the specified message number before using it in a command.

%message 1 "message 1"
%message 3 "message 3"  [ skipping message 2
print(2); [ NOT OK, message not declared
...
print(4); [ NOT OK, can't use message before declaring it
...
%message 4 "message 4"
...

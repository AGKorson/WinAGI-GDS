Logic Compiler Error 4028

Could not find include file '<filename>'

The compiler is not able to find the include file with the name '<filename>'.

If <filename> does not include an absolute or relative path, the compiler assumes it is in the game's resource directory and will look for it there.

Verify the filename is typed correctly, and that its path is correct (or the file is in the resource directory) to correct this compiler error.

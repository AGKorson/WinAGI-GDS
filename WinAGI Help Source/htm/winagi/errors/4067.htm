Logic Compiler Error 4067

Undefined word '<wordtext>' in said command

This compiler error is raised when a literal string passed as an argument to the said command is not found in the game's WORDS.TOK file.

It most likely indicates a typo. Check the spelling of the word in the said command to make sure it matches a valid word in your WORDS.TOK file.

Logic Compiler Error 4040

Syntax error at argument <number>: expected <type>

This error indicates the compiler was expecting an argument of the specified type, but a keyword, command name, or symbol was detected. This almost always is the result of a typographical error. Check your source code and adjust it to include the correct argument type.

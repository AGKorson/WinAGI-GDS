Text Screen Editor

To assist with creating text screen displays in you game (for example, help screens), WinAGI includes the Text Screen Editor. It provides a simple WYSIWYG (what you see is what you get) editor where you can add text and set colors to create text displays. You can then copy the necessary AGI commands (set.text.attribute, display) to create the text output in AGI which you can then paste into your logic source code. Text screen data can be saved to a file, which can later be opened for additional editing.

For a description of the various components of the Menu Editor window, click on the area of interest:

Text Screen Display

The text screen display shows an AGI text screen where text is displayed with the font and colors that are available in AGI.

When the entire display doesn't fit in the editing window, scrollbars are automatically displayed. You can also press the SHIFT key with the left mouse button to drag the screen as needed.

Text Markers

To assist with aligning text, small text markers appear at the lower right corner of each character position. You can toggle these marks off and on using the Show/Hide Text Marks command on the Edit menu.

Color Palette

The color palette is used to set the background and foreground colors of text. When typing text (or pasting unformatted text) the attributes that will be applied are indicated by 'B' for background and 'F' for foreground. Left click to select a background color, and right click to select a foreground color.

When text is selected, the palette changes to indicate the colors of the selection, 'b' and 'f'. If the selection has multiple colors, then no indicator is shown. Selecting colors by clicking will set the colors of the selection (the default colors are not affected, and will re-appear once the text is deselected).

Toolbar

Standard text edit commands as well as scaling commands are located on the toolbar.

Typing Mode: Overwrite/Insert

    When editing test, with INSert mode active, typing new characters or pasting will insert the characters, moving existing characters to the right. Any text pushed past the right edge is lost. OVeRwrite mode will replace existing characters as the cursor passes over them without moving existing characters. Backspace and Delete will also erase characters without affecting surrounding text.

Editing Tools

          Editing tools work just like any other text editor. You can select, copy cut or delete text. You can paste text copied from other editors or other programs.

When pasting, text copied from the text screen editor will include the text's back- and fore-ground colors. Text copied from another source will use the current colors.

The Clear tool is similar to the Delete tool, except it is only is enabled when a selection is made, and it only clears all the characters in the selection; it will not collapse the selection like the Delete key in Insert mode does.

NOTE: Currently, there is no undo capability in the text screen editor.

Copy as AGI Commands Tool

    After building the text screen, you can easily create logic code to recreate it in your game by using the Copy Commands tool. This tool will copy the necessary set.text.attribute, display commands to the clipboard. You can then paste them into your logic source code.

If there is no selection, the commands copied will build the entire screen. If a block of characters is selected, then only that block will get converted to commands.

Paste as AGI Commands Tool

If you have existing source code for displaying text, you can copy it and then use the Paste Commands tool to add it to the current display. The paste function will extract set.text.attribute and display commands and add the corresponding text in the designated colors to the display screen. Any text added will overwrite any existing text.

Any lines in the source that have errors will be ignored. Any lines that have invalid argument values will be ignored. The message argument in display commands must be a string value (can't be 'm##' or a define ). All text must be on one line - concatenated text will be ignored. Special format codes are also ignored.

Extended Character Support

   Although it was extremely rare to see in Sierra games, AGI does properly display all extended characters. Use the Character Map Tool (on the toolbar or Edit menu, or press CTRL+INS) if you want to add an extended character to the display.

Scale Control

    Use the scale controls to control the scale of the layout editor. The mouse wheel can also be used to adjust scale.

Status Bar

The status bar displays the current scale factor, as well as the mouse position (row/column). The status of the Shift, Num Lock and INS keys are also displayed.

Saving Text Screen Data to File

Use the Save Text Layout and Save As commands to save the current state of the text screen. The data will be saved in a custom format file with a 'tsd' extension. You can then use the Open Text Layout command at a later time to retrieve the text screen data and continue editing.

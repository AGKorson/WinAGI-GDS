Version Game Properties

Designer

You can use the designer field to store the designer's name. It is not used by AGI for any in-game purpose.

Description

A free text field you can use to record a description for your game. This property is used only by WinAGI; it is not used by or included in your compiled game.

Game About

A text field that can be used as a game description message for a game. It can be accessed in logic source code through a reserved define name. When logics are compiled, the reserved name is replaced with the Game About text.

Game Version

A text field that can be used as a game version message for a game. It can be accessed in logic source code through a reserved define name. When logics are compiled, the reserved name is replaced with the Game Version text.

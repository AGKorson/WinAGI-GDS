Picture Editor Text Test Mode

  The Print/Display test mode allows you to see what printed messages and displayed text will look like when displayed on your picture. When entering the Text Test mode, the Print/Display Test options dialog is automatically displayed:

Enter the text you want to see in the Message input box. Choose the command type (print, print.at, display) you want. Options for the selected type will appear on the right of the dialog box:

    • print - the basic print command has no options. The message box will be automatically centered horizontally and vertically on the picture.
    • print.at - the print.at command lets you set the location and maximum width of the message box. Available options are top row and left column of text, and the maximum width of the message box. See the print.at help topic for a detailed explanation of how these parameters affect placement of the message box
    • display - the display command prints message text directly on the screen, without a message box border and background. Options include the top row and left column of text, the desired foreground and background colors, and the picture row offset that you expect to use when the game is running. The offset is almost always 1, but you can choose any valid value. It is needed for the display command test because the starting row is the absolute screen row, not relative to the top of the picture.

Click OK to see the results. Press the ESC key, or left-click anywhere to dismiss/erase the text. You can select the mode again from the menu or toolbar, or double-click on the picture image to re-display the options dialog.

If you are satisfied with the results, you can click the Copy Command button to copy logic command code to the clipboard so you can paste it into a logic.

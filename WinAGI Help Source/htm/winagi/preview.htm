Previewing Resources

WinAGI provides a resource preview window for logics, pictures, sounds and views. (The OBJECT file and WORDS.TOK file can't be previewed.) The resource type and number, and the Resource ID are displayed in the Preview window title bar. You can hide or show the preview window by changing the corresponding setting on the General tab of the Settings form.

Double-clicking on the previewed resource will open the currently previewed resource for editing.

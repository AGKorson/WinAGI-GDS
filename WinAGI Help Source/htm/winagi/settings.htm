WinAGI Program Settings

WinAGI provides many customizable features that allow you to configure the development environment to suit your individual preferences. Use the Settings command on the Tool menu to display the Settings window.

(You can reset all settings to the original defaults by clicking the Default button.)

All settings can be modified by editing WinAGI's configuration file winagi.config, in a text editor, although this is generally not advised. The Settings form should be used to adjust program settings. The winagi.config file is stored in the program's main directory.

 

There are seven tabs on the Settings window:

General

General settings affecting overall program control.

Logics

Settings that control the logic editor's display, and syntax highlighting are on this tab.

Logics (Page 2)

WinAGI's compiler and decompiler settings, and the global editor settings can be modified here.

Pictures

Picture settings control the default behavior of the picture editor and picture preview.

Sounds

These settings control the default behavior of the sound editor.

Views

These settings control the default behavior of the view editor and view preview.

Layout

Layout editor display settings can be adjusted on this tab.

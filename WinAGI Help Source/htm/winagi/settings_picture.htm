Default Image Scale

You can set the default image scale factor for both the preview window and the picture editor. Use the drop down box to select one of the available scale options. The minimum scale is 100%. For the preview window the maximum is 1000%; the editor maximum is 2000%.

 

Drawing Surface Defaults

Split Window

When this option is enabled, the drawing surface is initially split in two when a picture editor is opened, showing both the visual and priority pictures. Drag the splitter to adjust the size of the two drawing areas.

Full Window

When the Full Window option is enabled, the picture editor drawing surface shows the visual picture only; the splitter is positioned at the bottom of the drawing surface. Right click the drawing surface to toggle between visual and priority pictures, or drag the splitter to shift back to split mode.

Show Priority Bands

When this option is enabled, the priority bands are turned on by default when a picture editor is first opened.

 

Coordinate Cursor Style

There are two options for cursor display when editing picture commands. The flashing box cursor highlights a selected coordinate's position with a small solid box that cycles through all colors. No other coordinates in the current drawing command are highlighted.

The 'X' cursor option highlights a selected coordinate with a small square outline which also cycles through all colors, and all other coordinates have 'x' markers to highlight their location. See the Cursor Options topic for additional information.

 

Default Picture Editor View Test Options

Drawing Pictures

The picture editor must be in the Edit mode to add drawing commands to the picture. Select the Edit mode by clicking the Edit mode toolbar button (or by disabling the test mode on the Resource menu).

 

Display Mode

The default display mode is step draw, which only draws the picture up to the currently selected command. When the drawing tool is either Select or Area Select, the currently selected command is also drawn, allowing it to be edited. If any other drawing tool is selected, all draw commands PRECEDING the selected tool are drawn; new commands inserted at this point will push the selected command down (i.e. they come BEFORE the current selected command).

You can switch to full draw mode by clicking the full draw toolbar button. In full draw mode, the entire picture is always drawn, regardless of which command is currently selected. This can be very useful to quickly check the effects of inserting a new command or editing an existing command.

 

Background

You can use the background tool to add a background image which can then be traced. Clicking the background tool button will toggle the background on and off. See the Using a Background Image topic for details on how to select and configure a background image.

Note that when you save an in-game picture, your background settings are saved as well.

 

While drawing, the main status bar will display useful information. The current image scale, mode, and selected tool are displayed in the first three panels. The current cursor position, and priority band of that position are displayed in the three panels to the right.

 

Drawing Tools

To add drawing commands to your picture, first select a pen color for the visual screen, or the priority screen or both.

The next step is to select a drawing tool. The drawing tools available in WinAGI are the Select, Area Select, Absolute Line, Relative Line, Corner Line, Rectangle, Trapezoid, Ellipse, Fill and Plot.

 

Select

The Select tool is the default tool. No drawing tool is active. With the Select tool, you can use the left mouse button to drag the drawing surface around, if it does not fit within the editor window.

If a drawing command that has coordinates is currently highlighted, the select tool allows you to edit individual coordinates. See the Editing Coordinates topic for additional information.

Also, selecting a coordinate in the coordinate list will always automatically switch to the Select tool.

 

Area Select

The Area Select tool is used to copy a rectangular area of the picture to the clipboard as a bitmap. Left click and drag to select an area. Use the Copy Selection command on the edit menu to copy the selected area to the clipboard.

The selected area is stored on the clipboard as a standard bitmap, and can be pasted in a view cel. This can be useful when you are making views that have to line up precisely with an area of a picture. Area Select the area where the view will go, then paste the area into your view. Now draw on top of the pasted data to create your view.

The anchor (starting) point of the area, and the entire block coordinates are also displayed on the status bar for reference. You can right click either of these status entries to copy the value as text to paste in a logic editor to easily add argument values to commands such as get.posn(), obj.in.box(), etc.

 

Absolute Line

Use this tool to draw lines of any direction and length. Once selected, left click on the drawing surface to place the starting coordinate. Continue left-clicking on new coordinates to draw additional line segments. Right-click on the drawing surface or select another tool to stop drawing.

 

Relative Line

For lines where the length of each segment is short, use the relative line tool. Set the starting point, and draw segments by left-clicking, the same as absolute lines. Each segment is limited to +/- 7 units in both directions from the previous point. WinAGI will automatically limit each segment for you to prevent causing an error. Right-click or select another tool to stop drawing.

 

Corner Line

Corner lines are a series of alternating horizontal and vertical lines. If the first segment is horizontal, the line is called an 'X Corner'; if it is vertical, it is called a 'Y Corner'. Left-click to set the starting point. Depending on how you move the mouse, the first segment can be horizontal or vertical. WinAGI will automatically restrict subsequent segments to the correct orientation. Right-click or select another tool to stop drawing.

 

Rectangle

The rectangle tool is a shortcut tool that makes it easy to draw squares and rectangles. Left-click to set one corner. Then move the mouse and left-click again at the opposite corner. WinAGI adds a Corner command that includes all four segments to create the rectangle.

 

Trapezoid

Use the trapezoid tool to easily draw floor and ceiling outlines. Left click to set one corner. WinAGI will automatically create a trapezoid centered in the picture using an Absolute Line command.

 

Ellipse

The ellipse tool is a very handy tool for drawing circles and ovals. Left click to set one corner. Move the mouse and left-click at the opposite corner. WinAGI will create four absolute line segments to draw the ellipse in four 90 degree arcs.

 

Fill

Use the fill tool to fill an area with a color. The behavior of the fill tool depends on current pen status:

      • if only the visual pen is enabled, the fill tool fills the visual screen in all directions from the starting point until a non-white color is encountered;
      • if only the priority pen is enabled, the fill tool fills the priority screen in all directions until a non-red (priority value of 4) is encountered;
      • if both pens are enabled, the fill tool will fill the visual screen in all directions from the starting point until a non-white color is encountered. The priority screen is filled to match the exact same shape as the visual screen, regardless of any boundaries on the priority screen.

Plot

Use the plot tool to add round or square brush strokes to a picture. The plot tool is used in conjunction with the Plot Shape and Plot Size tool dropdowns (located at the right end of the toolbar). To change the shape or size, choose the desired option from the dropdown. Make a new selection, and the main toolbar will update to indicate the newly selected plot effect. For detailed information about plot styles and sizes, see the Pen Size and Style command topic.

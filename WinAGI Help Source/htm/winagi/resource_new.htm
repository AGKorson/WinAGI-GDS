Creating New Resources

Use the New Resource command on the Resource menu, or the New Resource button on the main toolbar to create new resources.

New Logic/Picture/Sound/View Resource

    

If a game is NOT open:

Choosing to create a new resource will open a new resource editor. The resource can only be saved to a file.

If a game IS open:

After opening a new resource editor, the New Game Resource dialog is displayed.

Choose a resource number, resource ID, and an optional description. The default ID is 'Logic#', 'Picture#', etc. where # is the desired resource number. Click OK to finish creating the resource.

If you want to create a new resource, but do not want to add it to your game, click the New Not in Game Button on the New Game Resource dialog.

Create Matching Picture Option

For logic resources only, you can also choose to add a matching new picture to your game at the same time. (The available resource numbers will only include those where both logic and picture resources with that number are free.)

If choose to add a matching picture, both the new logic, and a new picture with the same number will be added to the game. If your new logic ID begins with "rm.", the matching picture will use the same id with a prefix of "pic." ("rm.Test1" becomes "pic.Test1" for example). Otherwise the ID will be the default "Picture<resnum>".

Include Room Template Code Option

With this option, boilerplate text for logics that is common for rooms will be pre-loaded into the logic upon creation. The boilerplate text can be edited if desired; the text is stored in a file called "deflog.txt" in the WinAGI game directory.

The boilerplate text can include any of these field codes:

      • ~ (tilde): indents a line using the current setting for tabs
      • %id: inserts the new logic's ID
      • %desc: inserts the new logic's description
      • %h: inserts the current horizon value as set in the Picture Editor's View Test Mode parameters

 

New OBJECT File, New WORDS.TOK file

There is not a button on the main toolbar for creating new OBJECT and WORDS.TOK file. Use the New Resource command on the Resources menu.

If a game is NOT open:

This command opens a new OBJECT file editor or WORDS.TOK file editor. You will be prompted to provide a filename when you save the resource.

If a game IS open:

After the new editor is opened, you will be asked if you want to replace the existing file with the new blank one; if you choose no, the open game's existing OBJECT or WORDS.TOK file is not affected; the new file just created will be treated as an object file not in a game. If you choose yes, the OBJECT or WORDS.TOK file is replaced with the new blank one you just opened.

Running Games

NOTE: WinAGI is not an interpreter. You need to have a compatible interpreter (Sierra, NAGI, AGILE, etc.) available to your game files in order to use the Run game feature.

You can run your game to test it by using the Run command in the Game menu. This feature only works if you have selected a platform on the Game Properties dialog. Currently, WinAGI supports the use of DOSBox (with original MSDOS Sierra AGI program files), ScummVM, AGILE, and NAGI.

Keep in mind that the selected platform will use the compiled logics that are currently in your VOL files; if any of your logic source files need to be compiled, you will have the option to compile the changed logics before the game is run.

WinAGI Help

WinAGI Game Development System is a full featured AGI game design and development system. It provides game designers with advanced editors and tools to easily create high quality AGI games.

This help file is designed to help designers take full advantage of WinAGI's powerful editors and tools.

 

Help is available in the following areas:

Working with Games

Help on how to work with AGI games, including help with creating new games, using template games, setting game properties, etc.

Working with Resources

WinAGI has editors that allow you to edit all resource types in AGI. This topic provides detailed information on how to use the resource editors, as well as basic resource management, such as creating new resources, saving, exporting or importing resources, adding to and removing from games, etc.

Working with Tools

WinAGI includes several tools to make game design and development easier. This topic provides detailed information on these tools, and how to use them to make game development easier and more efficient.

Program Settings

There are many user-configurable settings available to customize WinAGI. This topic provides detailed information on available settings.

Errors and Warnings List

Errors and warnings that are detected during the game design process are displayed on the Errors and Warning List.

Command Line Options and File Extensions

A description of command line options, and the various file types used in WinAGI and their associated file extensions.

Keyboard Shortcuts

A summary of all keyboard shortcuts in WinAGI GDS.

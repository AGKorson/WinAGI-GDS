The WinAGI Story

WinAGI was originally released in 2005 as a more modern alternative to AGI Studio. It was written in Visual Basic 6, and included improved resource editors and built-in tools to streamline the creation and editing of AGI games. It also included a detailed help file, with a detailed AGI reference section based on a full decompilation of the AGI engine and supporting utilities.

For two decades, several updates were released, which added additional features and more documentation of AGI's capabilities and limitations. But VB6, despite its robustness, was becoming unsustainable as a viable platform for further development.

In 2023 I decided to finally move WinAGI to a modern development platform. I chose C Sharp, because I felt it gave me the best mix of capabilities and ease of use. It also gave me the chance to gain proficiency with a more modern programming language. I knew it was going to be a massive project, because it would require a complete rewrite of the entire source code.

Although it has taken me several years, it is finally done. This new version of WinAGI builds upon its VB6 predecessor with a more modern interface, improved resource editing tools, and many enhancements that take advantage of features now available that weren't possible with VB6.

WinAGI is now well situated to be the premier AGI development suite well into the future. And the source code is available on Github, so it should be easy for anyone to improve and advance it going forward. I sincerely hope that that people find this to be a useful tool, and look forward to the next great fan-created AGI game that it helps bring to life!

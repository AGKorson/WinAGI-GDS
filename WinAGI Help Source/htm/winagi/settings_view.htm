View Settings

Default Scale

You can set the default scale factor for both the preview window and the view editor. The minimum scale is 100%; the maximum is 1500% for the preview window, 2000% for the editor.

 

Default New Cel Size

When a new cel is added to a view in the view editor, these values determine the initial height and width of the cel.

 

Default Preview Alignment

Use these drop down boxes to set the default alignment of views when they are shown in the preview window.

Default Pen Colors

You can adjust the default colors for the left and right mouse button when first opening a view editor.

 

Show Grid Lines

Grid lines are shown in the cel drawing area as an aid for pixel alignment. If you do not want grid lines to be displayed, uncheck this option.

 

Loop Preview

When this check box is enabled, the loop preview window will be open by default when a view is opened for editing. If you do not want previewing to be the default, uncheck this option.

 

Loop Cycling Enabled

When a view is opened, if the loop preview window is enabled, loops will cycle if this option is checked. If you do not want loops to cycle by default when the view is loaded, uncheck this option.

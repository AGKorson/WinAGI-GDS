Exporting Resources

Exporting a resource allows you to save a resource as a separate resource file outside of an AGI VOL file, or for some resource types, as a different file format.

Exporting a resource does NOT affect your game. It just makes a copy of it.

 

Export vs. Save As...

The Export command is similar to Save As... command. Both allow you to save a resource to a file.

The difference is that resources that are in a game can be exported. Resources that are NOT in a game can be saved as a different filename.

 

Exporting Logics

When you choose to export a logic, you are given the opportunity to export the actual compiled logic resource (*.agl files), the source code (*.lgc files) or both.

 

Exporting Pictures

In addition to exporting pictures as AGI resource files, you can also save the picture image as a graphic image in BMP, JPG, GIF, TIFF or PNG format.

When you select the Picture Image option, the dialog box expands to give you additional options for saving:

You can choose to export the Visual image, the Priority image, or both. If you choose both, WinAGI will add "_P" to the name you choose for the Visual image and use that for the priority image. You can also choose to scale up the image before saving.

You can also right click any picture resource in the tree list, and choose the Save Picture Image As... command on the Resource menu as a short cut to the export picture image dialog.

Export as Animated GIF: Pictures can be exported as an animated GIF that shows the picture being drawn step by step. Use the Export Picture as Animated Gif item on the Resource menu.

Options available include:

    • Scale: Sets the scale for the GIF. Note that the preview window does NOT indicate relative size; view the resulting GIF to determine if the scale chosen is satisfactory.
    • Continuous Loop: When checked, the resulting GIF will cycle continuously. When cleared, the GIF will cycle once, then stop on last cel.
    • Delay: Adjusts the time delay between cels in the GIF, in hundredths of a second increment. The preview window provides a reasonably accurate representation of the chosen delay time.

Export All Images: You can easily save the images for all your picture resources with a single command. When you select the Pictures node in the Resource List, you can use the Export All Picture Images command on the Resource menu. All picture images will be exported to your game's resource directory using the options you choose.

 

Exporting Sounds

When you choose to export a sound, there are four formats available:

    • AGI Native Sound Resource (*.ags) - this format stores resources as native AGI sound resources.
    • MIDI compatible conversion (*.mid) - WinAGI can save a sound as a MIDI file. Channels that are marked as muted in the preview window or in the editor window will not be in the output. Sounds can be saved as MIDI files, but WinAGI cannot read and convert MIDI files to AGI sound resources.
    • PCM/WAVE compatible conversion (*.wav) - WinAGI can also convert the sound to a wave file which emulates the sound output of the original IBM PCJr sound chip.
    • AGI Sound Script (*.ass) - this format saves a resource as a sound script. The sound script format has been established by Nick Sonneveld. WinAGI can read and write to sound script files.

 

Exporting Views

There are no other formats available to export view resources, however, view loops can be exported as animated GIFs. When a loop is selected in either the View Editor or the Preview Window, choose Export Loop as GIF command on the Resources menu.

Options available include:

    • Horizontal alignment: Adjusts the alignment of cels, if not all the same width, in the GIF. Choices are align right or align left.
    • Vertical alignment: Adjusts the alignment of cels, if not all the same height, in the GIF. Choices are align top or align bottom.
    • Scale: Sets the scale for the GIF. Note that the preview window does NOT indicate relative size; view the resulting GIF to determine if the scale chosen is satisfactory.
    • Transparency: When checked, the transparency color for the loop will be treated as transparent in the resulting GIF. If not checked, the transparency color will be included as a non-transparent color.
    • Continuous Loop: When checked, the resulting GIF will cycle continuously. When cleared, the GIF will cycle once, then stop on last cel.
    • Delay: Adjusts the time delay between cels in the GIF, in hundredths of a second increment. The preview window provides a reasonably accurate representation of the chosen delay time.

 

Auto Export Feature

The Auto Export feature automatically exports your picture, sound and view resources every time you edit them. You can enable or disable this feature on the General tab of the Settings window.

WinAGI uses the resource ID as the filename when auto exporting. If you change the resource ID, WinAGI will automatically change the name of the corresponding resource file, if one exists.

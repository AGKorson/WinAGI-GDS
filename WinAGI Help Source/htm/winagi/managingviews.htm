Managing Loops and Cels

Views are made up of a series of loops. Each loop is in turn made up of a series of cels.

A view must have at least one loop, and each loop must have at least one cel. WinAGI will not let you delete the last cel from a loop or the last loop from a view.

Loops

 

To add, delete, or modify a loop, first select a loop in the View Tree.

Use the Insert Loop command on the Edit menu to add a new loop. The new loop will be inserted ABOVE the currently selected loop. To add a loop to the end, select the Loop End marker.

    Delete the loop by using the Delete command on the Edit menu, or by using the Delete toolbar button. You can cut a loop, or make a copy of a loop that can be pasted in another view, or in a different position of the current view.

TIP: To change the order of loops in a view, cut the loop from its existing position, then paste it in the desired position.

Use the Clear Loop command on the Edit menu to automatically delete all cels in a loop, leaving a single cel one pixel wide and one pixel high.

Mirrored Loops

You can designate a loop to be a mirror of another loop by choosing the desired loop in the Mirror Loop property in the properties window. This will delete all cels currently in the loop, and replace them with the mirrored cels from its mirror pair. Editing cels in a mirrored loop will automatically update cels in its mirror.

If you want to unlink a mirrored pair of loops, select 'none' for the MirrorLoop property of either loop. After 'unmirroring', the cels in both loops will remain unchanged, and can then be edited separately.

Only loops 0 through 7 can be part of a mirror pair.

Version 2.089 does not support loop mirroring. If the view being edited is from a version 2.089 game, loops cannot be mirrored. If a view that has loops is already in a version 2.089 game (or added to the game), WinAGI will automatically remove the mirror property and save each loop independently when the view is saved in the View Editor.

Cels

 

To add, delete, or modify a cel, first select a cel in the View Tree.

Use the Insert Cel command on the Edit menu to add a new cel. The new cel will be inserted ABOVE the currently selected cel. To add a cel to the end, select the Cel End marker.

    Delete the cel by using the Delete command on the Edit menu, or by using the Delete toolbar button. You can cut a cel, or make a copy of a cel that can be pasted in another view, or in a different loop or position in the current view.

TIP: To change the order of cels in a loop, cut the cel from its existing position, then paste it in the desired position.

Use the Clear Cel command on the Edit menu to reset a cel to a one pixel wide and one pixel high cel.

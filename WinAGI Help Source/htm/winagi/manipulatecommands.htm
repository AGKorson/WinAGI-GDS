Manipulating Picture Drawing Commands

After a drawing command is added to a picture, it is very easy to edit, move, or delete them to fine tune your drawing. You can edit individual coordinates to reposition lines, fills and pen plots. You can also move, delete, copy, flip and paste whole commands or even groups of commands.

To manipulate commands and coordinates, the picture editor must be in edit mode; you can't manipulate commands in the test modes.

Editing Coordinates

If you want to reposition a coordinate in a drawing command, first select a command in the command list. Depending on the cursor mode, either select a coordinate from the coordinate list, or click on one of the highlighted coordinates on the drawing surface.

NOTE: You will see the active cursor on the visual surface only if the visual pen is active, and on the priority surface only if the priority pen is active.

Left-click and drag the coordinate to its new desired location. Keep in mind the following:

    • For Absolute Lines, Fills, and Pen Plots, coordinates can be moved anywhere on the screen.
    • For Relative Lines, coordinates are limited in movement so the resulting position does not exceed +/- seven units in both directions from any coordinates that precede or follow.
    • For Corner Lines, moving a coordinate will adjust the line so all segments remain horizontal or vertical.

For even finer control over coordinates, you can double-click on a coordinate in the coordinate list. This opens up a dialog box that allows you to enter the coordinate's X and Y values manually.

 

For plot commands, you can also manually select a splatter pattern for plot coordinates when the active pen is set to splatter.

 

The plot pixels will flash in the preview so you can see what's underneath them while you make adjustments to the plot settings.

There is a bug in AGI that uses 160 as edge limit when placing plot pixels. This causes pixels to wrap around to next row. There are a few original Sierra games that have this bug, so a warning is displayed when this condition is detected. WinAGI will not allow you to intentionally position a plot command if it would create this situation.

Inserting Coordinates

You can easily add coordinates to any existing command. Select a coordinate in the command's coordinate list and then use the Insert Coordinate item from the Edit menu or press SHIFT+INS. The inserted coordinate will be added immediately after the currently selected coordinate in the command's list with the same X and Y values. The new coordinate is automatically selected after adding, and can then be edited like any other coordinate.

Editing Commands

There are several tools and operations that can be used to manipulate commands. Select the command you desire to change by clicking on it in the command list. If you want to select more than one command, select the first command, then press SHIFT key and click on the desired ending command. You can also hold down the SHIFT key while using the up and down arrow keys to change the selected commands.

Changing Colors or Plot Pen Settings

If you want to change the color used for a set of commands, select the existing Set Pen command. Choose a new color on the color palette (left button for visual, right button for priority) to change the color.

To adjust plot settings, use the same approach; select the Set Pen command you want to change, then use the pen style and pen shape toolbars to choose new settings.

Moving Commands

You can easily move commands around the screen.

To move a single command, select the command's first coordinate. Hold down the Control key, and left-click to drag the entire command to its new location. WinAGI will draw a bounding box to indicate the extent of the selected command, and will automatically restrict movement so the command doesn't move outside the boundaries of the picture.

To move multiple commands, left-click and drag within the displayed bounding box. WinAGI will automatically prevent you from moving the commands outside the picture boundaries.

Cutting Commands

This copies the selected command(s) to the clipboard, then deletes them.

Copying Commands

Places a copy of the selected command(s) on the clipboard. The commands can then be pasted in another place in this picture, or in a totally different picture.

Pasting Commands

The commands on the clipboard are added at the current position. Once pasted, you can move the pasted commands to a new area of the picture as described above.

NOTE: When copying and pasting commands, if the commands on the clipboard do not include pen setting commands, the current pen at the insertion point will determine the size and shape used for the inserted commands. This sometimes causes unexpected results. Just make sure to include pen settings when copying commands, if possible, or insert additional Set Pen commands in front of the pasted commands.

Deleting Commands

Removes the currently selected command(s). Watch for unexpected results; for example, deleting a line that a future fill command uses as a border will cause the fill to spill over into other areas.

Flipping Commands

  The vertical and horizontal flip tools are a very handy set of tools that allows you to flip a command or group of commands.

As an example, suppose you want to draw a path with matching statues on either side. Draw a complete statue on one side. Then select all the commands that make up the statue. Click the copy commands button. move the cursor to the end of your picture. Now paste the selected commands. Drag them to the other side of the path, and then click the Flip Horizontal tool to finish.

Splitting Commands

Line, plot and fill commands that include multiple coordinates can be split to make it easier to manage editing your picture. To split a command, select the coordinate that you want to be the first coordinate of the new command, then use the Split menu command (there is no toolbar button). WinAGI will automatically adjust the picture commands, ending the selected command at the selected coordinate, and inserting a new command with the selected coordinate as the start point.

Joining Commands

You can also join commands. The commands must be sequential and of the same type; you can't join a relative line to an absolute line, or a plot to a fill. Also, for line commands, the ending coordinate of the the first command must be exactly the same as the starting coordinate of the second command. If the commands meet the criteria, the Join menu command will be enabled when you select the second command.

Clear Picture

To quickly reset a picture, erasing all drawing commands, use the Clear Picture menu item (there is no tool bar button).

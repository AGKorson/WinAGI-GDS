Resource List

The Resource List is only available if a game is loaded. You can choose a list tree (which shows all resources in a hierarchical structure), a list box (which shows only the resources of a selected type) or no Resource List by changing the List Type option on the General tab of the Settings dialog.

The Resource List provides an easy way to manage the resources that exist in your game. The Resource List also includes a property window that displays a number of properties for the selected resource.

Resource List Tree

The tree option for the resource list uses a tree structure to display all resources in your game. The top level node represents the loaded game. Each resource type is represented by the second level of nodes. Individual resources of each type are represented as child nodes under each resource type node.

By default, the caption of each resource node is its resource ID. You can also include the resource number in the caption (see the General Settings topic).

If you prefer not to use resource IDs in the list tree, you can select the Display Resources by Number option on the General tab of the Settings window. You can select the format of the displayed numbers as well.

When you select a node in the list tree, the property window will display the properties associated with the resource. For logic resources, if the source code has changed since the logic was last compiled, it will be displayed in the list tree with red text.

Selecting a resource will display it in the Preview Window, if the Preview Window is visible. Properties of the selected resource are displayed in the property window just below the list tree. You can double-click a resource in the list tree to open it for editing. You can also right-click a resource to display a context menu that allows you to select a number of typical resource commands.

Resource List Box

The list box option consists of a dropdown selection box, and a list box. In the dropdown selection box, you can choose the root level (by GameID), or one of the six main resource types.

For Logics, Pictures, Sounds and Views, the list box is populated with all the resources of the chosen type. As with the list tree, the resources can be displayed by resource ID, or by resource number.

Selecting a resource will display it in the Preview Window, if the Preview Window is visible. Properties of the selected resource are displayed in the property window just below the list box. You can double-click a resource in the list box to open it for editing. You can also right-click a resource to display a context menu that allows you to select a number of typical resource commands.

Navigation Bar

At the top of the resource list is the navigation bar. Each time you select a resource or header item in the resource list, that item is added to a stack that you can browse by clicking on the 'Back' and 'Forward' navigation buttons. This makes it very easy to quickly move back and forth among resources that you are working with.

To browse even faster through the list you can right-click and hold either navigation button to display a list of resources on the stack. Move the cursor up or down and the display will automatically scroll through the stack. Release the button on a resource to select it.

Property Window

Properties associated with the resource currently selected in the resource tree are displayed in the property window. You can adjust the number of rows displayed by dragging the splitter bar between the property window and the resource tree. Some properties can be edited directly in the property window. Others are either read only properties, or can only be edited in a resource editor or the Game Properties dialog.

Custom Tools

The Tools menu has six menu items that can be customized to run your favorite tools, link to AGI related websites, or link to documents related to your game.

When you choose a custom menu item, the action taken depends on what the menu item is linked to:

  • If the menu is linked to an executable application, clicking the menu item will run the application.
  • If the menu item is a website or url, clicking the menu item will open your preferred browser to the selected page. Make sure the target begins with the characters "http" or "www".
  • If the menu item is a document, clicking the menu item will open the default editor/viewer for the file, and open the file.

 

Customizing Tools

Use the Customize Tools option to display a dialog for managing custom tools:

 

Double-click in a cell to edit its contents. Text entered in the caption column is what will be displayed on the Tools menu.

In the target column, enter the path and name of a file, application, program, url or website, or click the browse for file button (the ellipsis) and select a file.

For programs and files, WinAGI uses the game directory as the current path if a game is loaded, or the program directory if no game is loaded. You can use relative paths (i.e. '..\') or the folder constant '%PROGDIR%' which represents the path to WinAGI's program directory. If no path is specified, then WinAGI uses the current path as described previously.

To remove an item, select the entire row by clicking on the row number and press the Delete key, or delete either the Caption or the Target value.

You can change the order of your custom tools by dragging one row to another.

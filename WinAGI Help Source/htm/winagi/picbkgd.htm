Using a Picture Background Image

The first time you click the background tool, you will be prompted to select an image. Once you have selected an image, the Background Configuration dialog box appears:

 

 

 

Load Image

Use the Load Image button to select a new background image. Once selected, the image will be shown in the preview window, and will default to Stretch to Fit, with transparency of 50%. Any valid image file can be used as a background, including jpeg, bitmap, gif, png, etc.

 

Stretch to Fit

Clicking this button will automatically resize the background image to fit within the picture drawing window, in other words, to 320 pixels by 168 pixels.

The example image display will adjust as well.

 

Full Size

The Full Size button will resize the background image to its full size. This is useful if you have a portion of the image you want to use as a background that is already the size you want.

The example image display will adjust as well.

 

Transparency

To make it easier to trace your background image, you can set the transparency of your drawn picture so the background image shows through your drawn picture.

After you've drawn some parts of your picture, and especially after you start filling in areas with the Fill tool, it might get difficult to continue tracing. You can use the transparency feature to let the background show through drawn pixels.

The default value for transparency is 50%.

To adjust transparency, slide the control button up or down. The example display will automatically adjust to give you an idea of how the transparency will appear in the editor.

 

Display Image Scale

To assist in fine tuning the placement and size of the background, you can choose one of three scale values (100%, 200%, 300%). Scrollbars are provided so you can view the entire image area while making adjustments.

 

Background Options

You can choose to include the background image in the visual image, priority image or both by using the provided check boxes.

There is also an option to treat the default colors (white on the visual screen, red on priority screen) as always transparent. Depending on the situation you may find it easier to do tracing with this setting on or off as needed.

 

Example Image Display

Your draw image, in its current state, will display as an overlay on top of the background image. The picture image is bordered by a dashed line to make it easier to see how it will relate to the background.

The size of the background image and position of the picture can be adjusted to get the desired alignment. Click and drag within the dashed line that represents the picture image to change the position of the picture relative to the background. Or click and drag the background outside the dashed line to move the background image.

To resize the background, click and drag the right or bottom edge of the image.

When you are satisfied with the size, position and transparency of the background, click OK to accept the changes.

Note that when you save an in-game picture, your background settings are saved as well. The next time you open the picture for editing, the background will also be loaded according to the previous settings. If the background image file is no longer available, the background settings get reset.

You can remove a background image and reset background settings to default at any time by using the 'Remove Background' option on the Edit menu.

Platform Game Properties

Run Game Platform

WinAGI is a game editor, NOT an interpreter; if you want to run an AGI game, you will need to find an interpreter or an emulator. If you do have an interpreter platform, you can configure WinAGI to automatically use that platform to run your game in order to test it. Currently, WinAGI supports the following platforms:

DosBox: DosBox is not an interpreter, but with it, you can run your game using an original Sierra MSDOS interpreter. When you select this option, you will need to designate the actual DOS executable file to run in DosBox (usually a loader file such as 'sierra.com', or an unencrypted interpreter 'agi.exe' file). WinAGI will automatically include the correct command line parameter to run the game in the current game directory. Please note that you must have a Sierra MSDOS interpreter with all required support files to run your game in DOSBox.

ScummVM: If you have ScummVM installed, you can configure WinAGI to use it to run your game during testing. WinAGI will automatically include the correct command line parameter to run your game.

AGILE: AGILE is an AGI engine written in C# by Lance Ewing.

NAGI: You can use NAGI to run a game during testing, but it is not as flexible as other platforms. NAGI will only run correctly if the NAGI program files are actually located in the game directory, so if you select this option, WinAGI will automatically identify the executable file as being "n.exe" in your game directory. Also, NAGI does not accept command line options, so that field is disabled when NAGI is selected.

If you have an emulator platform other than one of these, use the 'Other' option. It will be up to you to manually enter the correct information in the Command Line Options field in order to correctly run your game.

Platform Executable File

You must specify the actual executable file in order to run a game on a particular platform. Use this option to find the file and select it. Note that when you change platforms (by clicking on a different option button), this field and the options field is automatically cleared (unless you are choosing the 'other' option) to make sure the correct executable file gets identified for your chosen platform.

Platform Command Line Options

This field allows you to add extra command line options that will we included when WinAGI starts the selected platform to run a game. For information on a particular platform's command line options, see that program's documentation. NAGI does not accept command line options, so this field is not available when NAGI is selected as a platform.

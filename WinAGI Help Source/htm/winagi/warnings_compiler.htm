Logic Source Compiler Warnings

Even when WinAGI is able to successfully compile a logic, it may detect situations which are likely to cause errors at runtime, or that might cause the AGI engine to crash. If the error detection level is set to Medium, WinAGI will display warnings in the Errors and Warnings List after compiling.

Compiler warnings beginning '5xxx' apply to all logics regardless of syntax being used. Warnings beginning '7xxx' apply only to logics using original Sierra syntax.

For detailed information on syntax rules, see the AGI Fan Syntax and Sierra Syntax topics.

Where warning messages include dynamic text, such as an unknown command, or argument values, the dynamic text is represented by '<' and '>' characters, for example '<definename>'.

Resource Load Warning RW24

Unexpected WORDS.TOK end of file

This warning indicates that while extracting words from the WORDS.TOK file, the end of the file was reached when WinAGI expected more data. This usually indicates a corrupted file.

WinAGI will return all words that were successfully extracted before the problem occurred. You should check the word list closely, and re-insert any that may be missing. Saving the file in WinAGI will clear this warning.

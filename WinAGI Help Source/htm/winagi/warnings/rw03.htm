Resource Warning RW03

Unable to create resource directory '<directory>'

When importing a game, WinAGI attempts to create a resource directory to store logic source code files. This error indicates that WinAGI was not able to create the directory.

WinAGI will use the game directory as the resource directory.

Check that the folder where the game files are located are not write-protected, or that there is any other disk related issue with the folder.

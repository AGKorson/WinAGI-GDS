Resource Load Warning RW06

Picture <number> has at least one invalid color assignment - picture may be corrupt

AGI has a minor bug in the Set Visual Pen draw command; if the value passed is > 15, the priority pen value gets overwritten. The picture will draw, but priority color will not be as expected.

The invalid visual pen command doesn't cause a problem, but it should be removed. The invalid draw command will be highlighted in red in the Picture Editor draw command list, making it easy to spot.

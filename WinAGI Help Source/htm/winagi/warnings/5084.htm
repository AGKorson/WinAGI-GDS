Logic Compiler Warning 5084

Attempting to play a sound that does not exist will cause a trappable error

Attempting to use the sound command to play a sound that does not exist will raise trappable error #9. It is possible to recover, but that depends on the game programmer writing good error handling code.

NOTE: WinAGI can only detect attempts to access a non-existent sound; it can't detect cases where your code attempts to access a valid sound that hasn't been loaded. That will also raise trappable error #9.

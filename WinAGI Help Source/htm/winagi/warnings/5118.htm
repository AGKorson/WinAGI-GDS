Logic Compiler Warning 5118

Menu item "<itemtext>" text width exceeds menu display width

AGI uses the length of the first item in a menus to establish the width of the display box that holds the menu items. If a later item in the list is longer than the first item, its text will be drawn past the right edge of the display box. You should pad the first items with spaces so that it is at least as long as the longest item in the menu.

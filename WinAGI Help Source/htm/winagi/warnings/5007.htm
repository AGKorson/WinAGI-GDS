Logic Compiler Warning 5007

String argument value is not within expected range (0 - <number>)

Depending on which version of AGI you intend to use to run your game, the number of strings is limited to 12 or 24. WinAGI uses the Target Interpreter Version property of your game to determine if a string number is valid or not. AGI does not validate the string number at runtime, so you should avoid invalid string numbers.

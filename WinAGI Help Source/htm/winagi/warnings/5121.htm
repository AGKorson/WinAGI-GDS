Logic Compiler Warning 5121

'rol' (word group 9999) is not recognized as a special word group in version 2.089

Support for the special word group 9999 (usually represented by the word "rol") was not included in AGI's said command in version 2.089. It was added beginning with version 2.272.

If you use the word "rol" in a said command running in a version 2.089 game, it will not match input as expected. The said command will treat the "rol" word as any other normal word, and will only match if the player actually types 'rol' as a word on the input line.

[ if player types "LOOK AT TREE"
said("look", "rol")) will NOT return TRUE

[ if player types "LOOK ROL" 
said("look", "rol") will return TRUE

Logic Source Decompiler Warning DW15

Controller number is greater than maximum usable value for the target game version (<value>)

Depending on which version of AGI you intend to use to run your game, the maximum number of controllers is 29 or 49. This warning is raised when the controller number in a command is greater than the maximum. AGI does not validate controller numbers at runtime. Using an invalid number will have unexpected results.

You should check the decompiled code carefully, as the invalid controller value may indicate a corrupt logic resource. It may also be an error in the original source code that wasn't detected when the logic was compiled.

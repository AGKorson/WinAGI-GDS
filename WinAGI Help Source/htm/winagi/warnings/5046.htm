Logic Compiler Warning 5046

If HEIGHT is less than 2, AGI resets it to 2 automatically

When setting up AGI's internal trace feature with the trace.info command, AGI will automatically reset the value for HEIGHT to 2 if you pass a value less than that. (This is one of only a few cases where AGI validates input values. Yes, it is surprising that there are not more cases where AGI validates input to avoid errors and crashes.)

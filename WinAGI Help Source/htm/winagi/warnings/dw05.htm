Logic Source Decompiler Warning DW05

Non-unique inventory item '<itemtext>' at resource index <pos>

This warning is raised when the decompiler detects more than one inventory item in the game's OBJECT file with the same name text.

There are some Sierra AGI games known to intentionally have multiple items with the same name. But multiple same namr items could also mean there is a problem with the game's OBJECT file. It does not create an error condition. It does mean that your source code must refer to the item by its number, not its name; WinAGI will adjust the decompiled source code automatically.

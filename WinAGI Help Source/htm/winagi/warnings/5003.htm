Logic Compiler Warning 5003

Ambiguous inventory item for argument <number>

If you have more than one inventory item with the exact same name, WinAGI will not know which one you are referring to. This warning let's you know that a duplicate inventory item name has been used in your logic, and that WinAGI assumes you mean the first inventory item that uses that name. If you want to make sure the right inventory item is referenced, use the 'i##' notation to identify the item.

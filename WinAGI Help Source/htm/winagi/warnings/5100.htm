Logic Compiler Warning 5100

COLOR value other than 0 is treated as 15

When used on the graphics screen, both clear.lines and clear.text.rect commands can only clear the screen to either black or white. If the COLOR value passed in the command is any value other than black, AGI will use white to clear the area. Note that in the text screen, the COLOR value is completely ignored, and the area is always cleared to black.

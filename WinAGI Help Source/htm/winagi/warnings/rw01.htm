Resource Warning RW01

<restype> portion of DIR file is larger than expected; it may be corrupted

This warning indicates a version 3 directory file is not formatted as expected. No portion of the file should be larger than 768 bytes (3 bytes per resource, maximum of 256 resources). There are a few known Sierra games and demos that have extra unused data included that don't interfere with the valid resources in the file. Rebuilding the VOL files should fix this error by dropping all the unused data.

The warning may also indicate a corrupt directory file. If that is the case, there will most likely be many other resource errors to provide indication of a corrupt directory. It may not be possible to open/edit game resources if that is the case.

Resource Load Warning RW25

Ignored characters detected in one or more words

AGI's input parser ignores several characters ()'`-" ASCII values 39, 96, 45, 34) and treats several others as spaces (,.?!();:[]{} ASCII values 44, 46, 63, 33, 40, 41, 59, 58, 91, 93, 123, 125), so those characters can never be matched in words in the WORDS.TOK file. This warning indicates one or more words have ignored characters.

Sierra's tool for compiling WORDS.TOK files (WORDS.EXE) file did not perform error checking, and would allow programmers to add words with these characters even though they can't be matched. There are several Sierra games that have words with these characters.

The affected words do not cause any errors, but they should be removed, as they serve no purpose and waste space in the file.

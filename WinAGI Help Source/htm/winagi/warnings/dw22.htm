Logic Source Decompiler Warning DW22

No test commands in 'if' or 'OR' block at resource index <pos>

This warning indicates an if or 'OR' block was found with no test commands. Although AGI allows this when processing a logic at run time, it is bad coding practice and should be corrected.

This warning may indicate the logic is corrupt. You should examine the decompiled source carefully and make changes as necessary.

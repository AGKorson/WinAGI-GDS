Resource Load Warning RW07

Picture <number> has at least one invalid command byte - picture may be corrupt

While processing picture data, if AGI expects a draw command (byte value of 240 to 250), but the byte value is outside that range, AGI will ignore the byte.

The invalid draw command doesn't cause a problem, but it should be removed. The invalid draw command will be highlighted in red in the Picture Editor draw command list, making it easy to spot.

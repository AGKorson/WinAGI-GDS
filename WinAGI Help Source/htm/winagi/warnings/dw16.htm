Logic Source Decompiler Warning DW16

Word argument number is greater than maximum usable value (10)

This warning is raised when the word argument value in a word.to.string command is greater than 10 which is the maximum usable value. AGI does not validate the word number at runtime. Using an invalid number will have unexpected results.

You should check the decompiled code carefully, as the invalid word value may indicate a corrupt logic resource. It may also be an error in the original source code that wasn't detected when the logic was compiled.

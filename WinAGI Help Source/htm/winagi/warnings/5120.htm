Logic Compiler Warning 5120

One or more characters are not supported in this code page

All text in AGI resources is stored as byte value characters from the original MSDOS code page (usually 437 but other 8 bit code pages are possible). When compiling, WinAGI will convert the text in the editor (in the system's default code page) to the game's code page. This warning is raised if any of the characters in the source code are not valid for the current code page.

All invalid characters are converted to '?' characters when added to AGI resources.

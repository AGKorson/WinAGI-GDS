Logic Compiler Warning 5036

Variable is being assigned to itself

Your code uses the assignv command (or equivalent alternate syntax) to assign a variable to itself (i.e. "assignv(v99, v99);", or "v99 = v99;". While this does not cause a runtime error, it does nothing except waste memory and interpreter cycles. You should check your code to make sure you didn't mean to assign something else and just made a typo.

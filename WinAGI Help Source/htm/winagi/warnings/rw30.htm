Resource Load Warning RW30

globals.txt file has one or more invalid entries

This warning indicates one or more define entries in the globals.txt file are invalid. Either the define name or define value would cause an error if they are used in a logic. Examples include a define name with invalid characters, a duplicate name, values that aren't valid numbers or argument markers, etc.

You can open the file in a text editor to examine it, and make any needed corrections. Items that have an error will be highlighted in red bold text.

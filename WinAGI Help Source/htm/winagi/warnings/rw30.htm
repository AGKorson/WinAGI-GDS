Resource Load Warning RW30

Invalid data in WORDS.TOK at position <pos>

This warning indicates a word tried to copy characters from a previous word that exceeds the previous word's length when extracting words from WORDS.TOK. WinAGI ignores the invalid length value and will continue extracting words from the file if possible. The location of the error is provided if you wish to examine the file in a hex editor.

The WORDS.TOK file is likely corrupt. Check the list of words carefully and re-enter any missing words. Saving the file will clear this warning.

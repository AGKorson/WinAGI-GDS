Logic Compiler Warning 5068

The message box window may not draw correctly if COL < 2 or (COL + MAXWIDTH) > 39

AGI does not validate the values passed for COL and MAXWIDTH to the print.at command. When drawing a message box, a red border is included which takes up one row space above the message, one row space below, one column width to right and one column width to left. The actual width of the message box is determined by AGI based on how it splits the message to fit over multiple lines. If the split results in the right border being greater than 39, or if COL is less than 2 there won't be enough room to draw the border correctly, which will result in distorted and misaligned borders that affect the entire screen. See the print.at for additional details.

Logic Compiler Warning 5087

<command> has no functionality in MSDOS AGI interpreter

There are several AGI commands that were not intended to run on the MSDOS platform. The MSDOS interpreter calls a null function when one of these commands is encountered. For those commands that have no arguments, including init.disk, hide.mouse, and show.mouse, there is no effect, and AGI will continue processing with the next command.

You should remove these commands, unless you intend to run your game with a non-MSDOS AGI interpreter. Just keep in mind that WinAGI isn't designed to support that.

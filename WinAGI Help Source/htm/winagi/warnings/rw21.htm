Resource Load Warning RW21

First item is not the null '?' item

The OBJECT file in almost every Sierra AGI game includes a 'null' item (a single question mark character) as the first item in the list of inventory objects. The format of the OBJECT file does not require that to be the case. It is OK to assign any text string to the first item. There is at least one AGI game released by Sierra that actually did assign a non-null value to the first item.

If you intentionally choose to use a non-null item for item 0, you can safely ignore this warning.

Resource Load Warning RW21

Invalid text pointer encountered in OBJECT file

This warning indicates that WinAGI detected a text pointer entry in the OBJECT file's index table that is beyond the size of the file. If the inventory item with the invalid pointer is accessed (by using one of the inventory item logic commands) AGI will use an invalid memory location, leading to unpredictable results.

This usually means the file is corrupted.

WinAGI will treat the item as a null item. To clear the warning, open the file in WinAGI, make any desired changes, then save it.

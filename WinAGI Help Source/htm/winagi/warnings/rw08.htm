Resource Load Warning RW08

Picture <number> has extra data past the end of resource

Picture resources must always end with a DRAWING END byte (0xff). AGI needs that to know when to stop processing drawing commands. Once the end of drawing is reached, AGI stops processing data from the picture resource. If there are extra data following the DRAWING END byte, the extra data will always be ignored. They just waste space in memory, and should be removed.

Opening and saving the picture in WinAGI will fix this anomaly.

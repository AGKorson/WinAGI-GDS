Logic Compiler Warning 5090

Undefined message (<messagenumber>); null value assumed

A message argument in form 'm##' was detected, but there is not a corresponding message preprocessor declaration. If the message index is greater than the last declared message, AGI will erroneously access an invalid memory location. If the message number is less than the last declared message number, AGI will raise trappable error #14. You should check your code for typographical errors.

Logic Source Decompiler Warning DW13

Screen object value is greater than game's MaxScreenObj (<value>)

When decompiling a logic, if WinAGI detects a screen object number used as an argument in a command that exceeds the loaded game's MaxScreenObj value, this warning is displayed.

This could indicate a corrupted logic resource, or it could be from a coding error when the logic was originally written. It may also occur if you reduce the MaxScreenObj value without adjusting affected source code and recompiling.

The logic will run, but when AGI tries to use the invalid object value, it will cause a trappable error (in some cases) or will try to access invalid information leading to unexpected results, including crashing AGI.

You should examine the decoded logic and make any adjustments needed.

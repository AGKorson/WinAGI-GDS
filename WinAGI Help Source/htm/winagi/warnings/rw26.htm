Resource Load Warning RW26

One or more duplicate words encountered

This error indicates that one or more duplicate words were found when the WORDS.TOK file. They don't cause an error in AGI, but only the first instance will ever be matched.

Sierra's tool for compiling WORDS.TOK files (WORDS.EXE) prevents the inclusion of duplicate words, but there may be fan created tools that don't.

WinAGI will automatically delete the extra words. Saving the file will clear the warning.

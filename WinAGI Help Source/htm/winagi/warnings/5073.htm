Logic Compiler Warning 5073

X1 should be <= X2 and Y1 should be <= Y2; this test command will not behave as expected

AGI expects coordinates used in test commands related to screen object position that define a rectangular area to be in 'upper-left-to-lower-right' format. If your coordinates are backwards, the test will never return TRUE, even if the object you are testing is within the bounds you defined. You should edit your source code so the coordinates are in 'upper-left-to-lower-right' format.

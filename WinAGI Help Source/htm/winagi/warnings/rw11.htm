Resource Load Warning RW11

Sound <number> has one or more zero-length notes

If the length parameter of a note in a PCJr sound resource is set to zero, it will be immediately skipped, with no sound output. It doesn't cause an error, but it is not necessary.

When WinAGI detects a zero-length note, it automatically ignores the note when opening the sound for editing. Saving the sound will then clear this warning.

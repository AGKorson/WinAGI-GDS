Logic Compiler Warning 7004

Reserved <flag/variable> is normally read-only; changing it could have unexpected results

Only applicable when using Sierra syntax.

Several of the reserved variables and reserved flags are used by AGI to manage game state. Using them in commands where they may be altered can lead to unexpected behavior. Unless you are intentionally modifying it for a specific reason you should avoid using the indicated variable or flag in commands where it may be altered.

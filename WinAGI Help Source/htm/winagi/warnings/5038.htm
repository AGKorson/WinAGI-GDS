Logic Compiler Warning 5038

Position is off screen (X > 159 and/or Y > 167)

Using values for position commands that are outside the screen coordinates will not cause a runtime error. But AGI will use the shuffle function to determine the object's final position before drawing it to the screen, and this will almost always result in unexpected positioning of objects. You should review your code and adjust position values to be within screen coordinates.

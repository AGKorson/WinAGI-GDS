Logic Compiler Warning 5115

submit.menu called before any menus added

All menus and menu items mst be added before calling submit.menu. This warning indicates a submit.menu command was encountered before any set.menu commands. Since no menus have been added, no menu can be displayed. You should check your source code and confirm that all set.menu and set.menu.item commands are called before submit.menu.

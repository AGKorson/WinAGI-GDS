Logic Compiler Warning 5071

Using a value > 167 for set.pri.base results in entire screen being priority 4

The set.pri.base command is used to adjust the bands that determine screen priority. It is rarely used, but if you do want to change the priority bands, using a value greater than 167 will make the entire graphic screen a priority of 4. If this is not what you intended, check your code for errors.

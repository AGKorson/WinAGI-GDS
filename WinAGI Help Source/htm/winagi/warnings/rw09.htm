Resource Load Warning RW09

Plot and PlotStyle draw commands are not valid in versions 2.089 and 2.272

The plot and plotstyle draw commands were not available in picture resources until version 2.411. This warning indicates this picture is in a version 2.089 or 2.272 game and contains one or more plot and/or plotstyle draw commands. The commands and their parameters are ignored by AGI and don't cause a problem, but they should be removed. The invalid draw commands will be highlighted in red in the Picture Editor draw command list, making it easy to spot them.

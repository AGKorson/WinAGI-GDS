Logic Compiler Warning 5085

Using add.to.pic command with an invalid loop will raise a trappable error

When processing the add.to.pic comand, AGI checks to see if the loop value is valid (by attempting to load the view and loop). If the loop is not valid, AGI will raise trappable error #5. It is possible to recover, but that depends on the game programmer writing good error handling code. You should edit your code so that the loop value is valid.

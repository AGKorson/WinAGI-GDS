Logic Compiler Warning 5049

INPUTLINE and/or STATUSLINE are inside the graphic display area; this is usually not desired

When the input line (where user enters commands) and/or the status line (where score and sound status are displayed) are in the graphic display area, they will be covered up anytime objects move through them. Unless you specifically want this effect, you should edit your code so input and status are on different lines. See the configure.screen command for additional information.

Logic Source Decompiler Warning DW04

Reference to null inventory item ('?') at resource index <pos>

Most of Sierra's AGI games use the question mark '?' to designate a null, or unused inventory item. The first item, i0, is usually designated as a null item. This warning is raised when a command that uses inventory items as an argument reference a null inventory item.

This is possibly due to a corrupt OBJECT file, or it could be due to a mistake in the original Sierra source code. It does not cause an error condition, but it is highly unusual to use the null item marker in AGI game source code.

You should confirm the OBJECT file is not corrupt, and also carefully examine the decompiled source code to determine if the offending code needs to be removed or corrected before recompiling.

'<word>' is a Group 0 word, which is always ignored by AGI, so it will never be detected by the 'said' command

Vocabulary words from group 0 are always removed from player input during parsing. Only non-zero group words get added to the word input array and can be checked by the said command. See the Reserved Word Group Numbers topic for additional information.

Logic Source Decompiler Warning DW14

String number is greater than maximum allowed value for the target game version (<value>)

Depending on which version of AGI you intend to use to run your game, the maximum number of strings is 12 or 24. This warning is raised when the string number in a command is greater than the maximum. AGI does not validate string numbers at runtime. Using an invalid number will have unexpected results.

You should check the decompiled code carefully, as the invalid string value may indicate a corrupt logic resource. It may also be an error in the original source code that wasn't detected when the logic was compiled.

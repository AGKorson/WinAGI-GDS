Resource Load Warning RW20

View <number> uses an unsupported loop mirroring format

In version 2.230, loop mirroring used a different format than the mirroring found in later versions. This warning is raised when a view in a version 2.230 game has mirrored loops using this format.

WinAGI does not support editing of views using this loop mirroring format. If you choose to edit and save this view, the mirrored loops will be replaced with separate, non mirrored loops. The view will look and work exactly the same, but its size will increase since the loops will no longer share data.

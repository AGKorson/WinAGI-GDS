Logic Compiler Warning 5104

Print message window may not display correctly if MAXWIDTH > 36

AGI does not validate the value passed for MAXWIDTH to the print.at command. The actual width of the message box is determined by AGI based on how it splits the message to fit over multiple lines, up to MAXWIDTH. If the assigned width of the message box is > 36, it will not draw correctly. You should check your code, and make sure you are using the correct value for MAXWIDTH.

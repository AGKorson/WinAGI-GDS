Logic Compiler Warning 5107

If LOWER = UPPER + 1, AGI will crash due to 'Divide by 0' error

The formula that AGI uses to calculate random numbers with the random command does not validate the input values before doing a divide operation. If LOWER equals UPPER + 1, a divide by zero error will occur; AGI will hang up and stop running. You should correct this error by using values of LOWER and UPPER that avoid this combination.

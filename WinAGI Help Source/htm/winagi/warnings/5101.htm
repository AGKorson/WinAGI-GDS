Logic Compiler Warning 5101

MGN values > 15 will cause unexpected results in add.to.pic command

The add.to.pic uses only the lower nibble (lower four bits) of the value passed by MGN. If the value is 3 or less, add.to.pic draws a control line; no control line is drawn if the value is > 3. If you pass a value to MGN that is > 15 expecting to get no control line, if the lower nibble of that number is 3 or less, you will instead get a control line drawn. Try to use a value <=15 for the MGN argument.

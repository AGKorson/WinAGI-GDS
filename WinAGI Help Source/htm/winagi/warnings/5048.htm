Logic Compiler Warning 5048

INPUTLINE and STATUSLINE are the same, which is usually not desired

When the input line (where user enters commands) and the status line (where score and sound status are displayed) are the same, every time the input line is cleared, the status line will be erased. And if the status line is displayed, entering commands will partially overwrite it. You should edit your code so input and status are on different lines. See the configure.screen command for additional information.

Logic Compiler Warning 5044

TOP argument is too large; portions of the play area will be off-screen, and graphics functions will be unpredictable

The value for TOP should not be greater than three, or the graphics window in AGI will be off screen. Values passed to the configure.screen are in text rows. Each text row is eight pixels high. The entire screen resolution is 640 x 200. This means there are 25 text rows available (8 pixels times 25 rows equals 200 pixels total). Like graphics screen coordinates, text row and column values are zero-based, meaning top row is row 0, bottom row is row 24.

Since the graphics window in AGI is 168 pixels (equivalent to 21 text rows), setting TOP higher than three will mean that some portion of the graphics window will be off screen, resulting in unpredictable behavior by AGI when drawing and displaying the graphics window.

Logic Compiler Warning 5086

Using add.to.pic command with an invalid cel will raise a trappable error

When processing the add.to.pic command, AGI checks to see if the cel value is valid (by attempting to load the view, loop and cel). If the cel is not valid, AGI will raise trappable error #8. It is possible to recover, but that depends on the game programmer writing good error handling code. You should edit your code so that the cel value is valid.

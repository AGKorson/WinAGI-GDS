Logic Compiler Warning 5060

Controller argument value is not within expected range (0 - 49)

Assigning a controller value greater than 49 will result in AGI overwriting memory used by other AGI processes, as well as unexpected behavior of the controller. When AGI evaluates key input, keys assigned to controllers above 49 will write data outside the controller table, which can interfere with sound playback and the internal priority table. Similarly, when using the controller test command with invalid controller values, AGI will read invalid data outside the controller table. See the Controllers topic for additional information.

Resource Load Warning RW10

PlotStyle draw command is ignored in version 2.411

Although plot and plotstyle draw commands were introduced in version 2.411, the plotstyle draw command had no functionality; its paramter is ignored and the pen is always set to a single pixel. This warning indicates this picture is from a version 2.411 game and contains one or more plotstyle draw commands.

The commands are ignored by AGI and don't cause a problem, but they should be removed. The invalid draw commands will be highlighted in red in the Picture Editor draw command list, making it easy to spot them.

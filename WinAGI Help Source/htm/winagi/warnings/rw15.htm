Resource Load Warning RW15

Loop <number> in view <number> shares cel data with another loop but is not a mirror

Loop mirroring in AGI is accomplished by having two loops use the same cel data, with each shared cel having a special bit set so AGI knows to flip the cel when the loop changes.

This warning indicates WinAGI found a loop that is sharing data with another loop, but the mirror bit marker is not set for any of the cels. Both loops will work without errors in AGI and will look exactly the same, but there aren't many practical reasons for having identical loops in a view.

Opening and then saving the view in WinAGI will update the view with correct formatting and will clear the warning.

Logic Compiler Warning 5108

The word '<wordtext>' is not in WORDS.TOK, but AGI automatically assigns it to group 0

When player input is parsed, the single letter words 'a' and 'i' are automatically assigned to word group 0 if they don't exist in WORDS.TOK. Most AGI games explicitly add them to word group 0, but it is not required.

Keep in mind that using words from group 0 in a said command will also generate warning 5083 to remind you that using words from group 0 in a said command will never return a match.

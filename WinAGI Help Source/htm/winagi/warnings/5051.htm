Logic Compiler Warning 5051

X2 - X1 should be >= 2 and Y2 - Y1 should be >= 2; this block will not behave as expected

AGI expects X2 to be greater than X1 by more than one pixel (and also Y2 to be greater than Y2 by more than one pixel. AGI does not check for this when you create a block with the block command. When the interpreter tests if any object attempts to cross into or out of the block, it will always return false for a block that is not set up correctly. You should check your source code and edit it so the coordinates are correctly passed.

Logic Compiler Warning 5113

Menu "<menutext>" has no items assigned to it

Menu items can only be added to a menu immediately after adding the menu. This warning indicates that no items have been added to this menu when another set.menu or submit.menu command was encountered, so the menu will remain empty. There is no practical reason for adding a menu with no items. You should check your source code and confirm the menu commands are arranged correctly.

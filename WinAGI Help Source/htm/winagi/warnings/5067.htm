Logic Compiler Warning 5067

The message box won't draw correctly if ROW > 22

AGI does not validate the value passed for ROW. When drawing a message box, a red border is included which takes up one row space above the message, one row space below, one column width to right and one column width to left. Therefore, if ROW plus the number of displayed lines of text exceeds 23, the message box won't draw correctly. Assuming a minimum line count of one, ROW must be <=22 (or lower if more lines of text are in the message box), otherwise the message box will be distorted and have misaligned borders that affect the entire screen. See the print.at for additional details.

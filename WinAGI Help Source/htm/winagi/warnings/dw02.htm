Logic Source Decompiler Warning DW02

Invalid word <wordnum> at resource index <pos>

If the decompiler detects a word group number in a said command that does not have a corresponding entry in the game's WORDS.TOK file, this warning is raised.

This could be caused by having the wrong WORDS.TOK file, or if the original Sierra source code had a mistake in it. Having an non-existent word group number doesn't cause an error- it just means this said command will never be able to return true.

You should confirm that you have the correct WORDS.TOK file. If the file is correct, you should carefully examine the decompiled source code to determine if the offending code needs to be removed or corrected before recompiling.

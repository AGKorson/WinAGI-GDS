Logic Compiler Warning 5093

Using control characters 8 or 13 in message text will result in incorrectly displayed messages

For most character values less than 32, AGI will display the appropriate glyph. But values of 8, 10 and 13 are spacing codes that will not display as a glyph, but will instead affect the cursor location for subsequent characters.

See the AGI Character Drawing Function discussion for additional information.

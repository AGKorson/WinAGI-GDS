Resource Load Warning RW17

View <number> has an invalid cel pointer for cel <celnum> of loop <loopnum>

The pointer for the specified cel points to a location outside of the view's data, meaning it would try to read cel information from an unknown location in memory. It likely indicates a corrupt view resource.

WinAGI will treat the cel as if it is a default cel with a height and width of one pixel.

To clear the warning, open the view, make any desired changes, then save it.

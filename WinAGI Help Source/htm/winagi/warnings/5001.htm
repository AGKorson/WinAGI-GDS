Logic Compiler Warning 5001

This command block contains no commands

WinAGI has detected an if block which has no commands in it. This does not cause an error in AGI during runtime, and it has no effect other than to waste memory. You should delete if blocks that don't contain any commands.

Note that if you want to use AGI Studio to decompile a logic which has an empty block, AGI Studio will identify it as a corrupt logic even though it really is not. It's best if you don't use AGI Studio anyway, since WinAGI is vastly superior in all respects.

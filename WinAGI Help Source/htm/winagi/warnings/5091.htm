Logic Compiler Warning 5091

Using m0 as a message value will cause an error

Message index numbers in AGI must be in the range 1 - 255. Zero (i.e. m0) is not a valid message number because of the way that AGI manages message indexing. If AGI tries to use a message with an index of zero, it will will erroneously access an invalid memory location and display meaningless data.

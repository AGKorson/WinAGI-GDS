Logic Compiler Warning 5040

Referencing a logic that does not exist will crash AGI

If you assign a logic to the trace.info command that does not exist, the next time a new.room command is called, AGI will try to load it. This will result in a runtime error and AGI will crash. Edit your code to point to a valid logic that contains the messages that trace.info will use for displaying trace information. See the Trace Mode topic for additional information.

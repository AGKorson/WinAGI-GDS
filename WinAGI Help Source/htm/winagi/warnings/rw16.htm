Resource Load Warning RW16

Loop <loopnum> of view <viewnum> has no cels

A properly formatted view must have at least one valid loop, and each loop must have at least one valid cel.

This warning indicates the specified loop in the affected view has no cels. If you try to assign this loop to a screen object, AGI will raise trappable error 8.

WinAGI will treat the loop as if it has a single cel with a height and width of one pixel.

To clear this warning, open the view in WinAGI, make any desired changes to the loop, then save it.

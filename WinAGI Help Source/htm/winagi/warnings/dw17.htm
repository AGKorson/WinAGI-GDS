Logic Source Decompiler Warning DW17

<type> command <number> not defined (<name>)

Only applicable when using Sierra syntax.

When adding a command using alternate syntax, the decompiler expects the declaration for the command to be the exact spelling (and all lower case) as indicated below:

Action Commands:

    • increment
    • decrement
    • assignn
    • assignv
    • addn
    • addv
    • subn
    • subv
    • lindirectn
    • lindirectv
    • rindirect

Test Commands:

    • equaln
    • equalv
    • lessn
    • lessv
    • greatern
    • greaterv
    • isset

This warning indicates the declaration in the game's sysdefs file is either missing or does not exactly match the expected spelling. The correct declaration will be added to this file, but you should check your sysdefs file and update it as needed.

Logic Source Decompiler Warning DW03

Invalid inventory item <itemnum> at resource index <pos>

If the decompiler detects inventory item value that is greater than the maximum number of inventory items in the OBJECT file, this warning is raised.

This could be caused by having the wrong OBJECT file, or if the original Sierra source code had a mistake in it. If AGI commands try to reference a non-existent inventory item, AGI will raise an error.

You should confirm that you have the correct OBJECT file. If the file is correct, you should carefully examine the decompiled source code to determine if the offending code needs to be removed or corrected before recompiling.

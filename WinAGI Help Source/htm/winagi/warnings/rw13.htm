Resource Load Warning RW13

View <number> has an invalid data pointer for loop <loopnum>

The pointer for the specified loop points to a location outside of the view's data, meaning it would try to read loop information from an unknown location in memory. It likely indicates a corrupt view resource.

WinAGI will treat the loop as if it has a single default cel.

To clear the warning, open the view, make any changes or corrections to the view, then save it.

Logic Compiler Warning 5052

If ROW >= 25 the prompt string is drawn at current text cursor location (both ROW and COL arguments are ignored)

If the ROW value is greater than or equal to 25, AGI will ignore both values, and display the prompt for the get.string at the current text cursor location. Note that if you pass an invalid COL value, AGI won't check for that, and the command will not behave as expected. See warning 5080 and the get.string topic for more details.

Logic Compiler Warning 5009

Low values for script size will most likely cause a trappable error

AGI uses script entries to track loading and unloading of resources, mainly to support saving and restoring. If the maximum script value is set too low by the script.size command, it is likely that as resources are loaded and unloaded, the script table will fill, and AGI will raise trappable error #11.

See the Scripts topic for a complete discussion on scripts.

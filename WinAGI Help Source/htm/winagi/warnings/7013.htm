Logic Compiler Warning 7013

said command word arguments will automatically be converted to lower case

Only applicable when using Sierra syntax.

The argument values in a said test command should be lower case because all words in WORDS.TOK are lower case.

However, to maintain compatibility with Sierra's logic compiler CG.EXE, WinAGI will automatically convert argument values with upper case characters to lower case.

Although using upper case characters in said command arguments does not cause an error, it is a bad coding practice. You should correct your code to use all lower case.

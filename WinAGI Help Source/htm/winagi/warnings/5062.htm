Logic Compiler Warning 5062

Target position is off screen (X > 159 and/or Y > 167)

AGI does not check that the values passed for target position for position commands are valid. If you pass X and/or Y values that are outside the graphics display area, the object will never reach its target; it will move to the closest edge and be stuck there. If you plan to use the flag that indicates the target was reached, you will need to edit your code to provide a reachable target.

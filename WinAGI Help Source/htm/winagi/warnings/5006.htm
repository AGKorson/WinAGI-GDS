Logic Compiler Warning 5006

Object number exceeds MaxScreenObject value (<number>)

Attempting to use a screen object value greater than MaxScreenObject generates a trappable error if the command is animate.obj, draw or erase. In all other cases, AGI will attempt to manipulate invalid data and will behave unexpectedly (and will likely crash).

Although you can successfully compile a logic with invalid screen object numbers in commands, you should never use invalid screen object numbers.

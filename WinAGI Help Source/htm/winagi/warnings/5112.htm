Logic Compiler Warning 5112

Using submit.menu command outside logic 0 wastes memory

Calling submit.menu outside logic zero wastes memory, because AGI resets its internal memory pointer, which results in the memory used by all resources currently loaded at the time the command is called to be behind the pointer. This can waste a lot of memory, which can easily result in AGI crashing due to exceeding memory requirements. See the Memory Management topic for detailed information on managing memory on the heap. The Menus in AGI topic has detailed information on managing menus.

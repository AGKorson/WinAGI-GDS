Resource Load Warning RW22

Abnormal word index table

The index of words at the start of the WORDS.TOK file has extra entries. Normally there are exactly 26 2-byte entries. At least one Sierra game is known to have extra entries in the index (for capitalized words, which are ignored by AGI). The extra index (and words) are ignored by AGI.

Recompiling (saving) the file in WinAGI will correct the problem.

This warning could also mean the WORDS.TOK file is corrupt. If so, there will most likely be other errors in the words to indicate this. The file is most likely unrecoverable.

Logic Source Decompiler Warning DW01

Invalid message <msgnum> at resource index <pos>

This warning means WinAGI detected a command that uses a message argument, but the message number used is one that does not exist in the logic. Usually this is because of orphaned code that was left in the original source that Sierra compiled.

You should carefully examine the decompiled source code to determine if the offending code needs to be removed or corrected before recompiling.

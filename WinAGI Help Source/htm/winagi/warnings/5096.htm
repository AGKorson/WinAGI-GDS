Logic Compiler Warning 5096

Using a long input prompt is unusual and will limit player input

The input prompt is typically a singl character, such a the ">" or "]". In some cases, you may want a longer string for an input prompt, but be aware that the amount of space available to the player to type input is limited by the space left on the line following the input prompt. If the prompt is too long, there may not be enough space for players to type the input you expect from them.

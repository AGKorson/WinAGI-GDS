Logic Compiler Warning 5103

Using a value of 1 for MAXWIDTH will crash AGI

AGI does not validate the arguments passed to the print.at command. Using a very small value for print width is unusual, and, due to a bug in AGI, using a value of 1 will actually crash AGI. (Using zero will result in a default width of 30.) You should check your code, and make sure you are using the correct value for MAXWIDTH.

Logic Compiler Warning 5063

The ROW and HEIGHT combination specified will not fit on the screen

AGI does not verify that the ROW and HEIGHT combination passed to the trace.info will fit in on the screen. If the ROW + HEIGHT value exceeds 23, the trace window will overflow the graphics display and not display the window correctly. You should modify your code so that the total of ROW and HEIGHT does not exceed 23.

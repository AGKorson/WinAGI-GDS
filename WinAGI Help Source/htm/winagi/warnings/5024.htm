Logic Compiler Warning 5024

Attempting to discard a view that does not exist will cause a runtime error

Attempting to discard a view that doesn't exist will raise trappable error #1. It is possible to recover, but that depends on the game programmer writing good error handling code.

NOTE: WinAGI compiler can only detect attempts to discard a non-existent view; it can't detect cases where your code attempts to discard a valid view that hasn't been loaded. That will also raise trappable error #1.

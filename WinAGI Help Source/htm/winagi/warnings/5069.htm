Logic Compiler Warning 5069

Invalid cursor position for clear.text.rect command; must be in upper-left/lower-right format

AGI expects values for text screen coordinates for the clear.text.rect command to be in upper-left-to-lower-right format. If the coordinates are backwards, AGI will not behave as expected, and may crash. Check your source code and adjust so the coordinates are in the correct order.

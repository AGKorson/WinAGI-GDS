Resource Load Warning RW18

View <number> has invalid cel data for cel <celnum> of loop <loopnum>

This warning indicates WinAGI was unable to create a valid cel from the data in the view resource. This most likely means the view is corrupted.

WinAGI will treat the loop as if it has a single default cel.

To clear the warning, open the view, make any changes or corrections to the view, then save it.

Resource Load Warning RW18

View <number> has invalid cel data for cel <celnum> of loop <loopnum>

This warning indicates WinAGI was unable to create a valid cel from the data in the view resource. This can occur if either the height or width is zero, or if the data is not properly formatted to create a valid cel. It most likely means the view is corrupted.

WinAGI will replace any cel data that are unreadable with transparent pixels.

To clear the warning, open the view, make any changes or corrections to the view, then save it.

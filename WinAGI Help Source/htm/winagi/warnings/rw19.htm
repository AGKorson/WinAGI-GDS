Resource Load Warning RW19

View <number> has an invalid view description pointer

For views that are used with the show.obj and show.obj.v commands, a view description text string is included in the view data. If the pointer to the description is invalid, AGI would display meaningless characters from whatever location in memory the pointer refers to.

WinAGI will ignore the bad pointer. Opening and saving the view in WinAGI will correct this anomaly.

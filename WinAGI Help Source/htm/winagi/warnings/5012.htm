Logic Compiler Warning 5012

Using room 0 in the new.room command will crash AGI

You should not use room 0 as the argument value for the new.room command. Since most games will use the call.v command in each cycle to run the logic code for the current room, this will result in AGI recursively calling logic 0 until the stack eventually gets blown and AGI quits.

Logic Source Decompiler Warning DW06

Unknowntest19 at resource index <pos> is only valid in Amiga AGI versions

All known version 2 and version 3 MSDOS AGI releases recognize eighteen test commands. There are claims that at least one Amiga release added a nineteenth command, possibly named in.motion.using.mouse, but this has not been confirmed. This warning is raised if WinAGI detects use of the nineteenth test command.

If this game is known to be an Amiga release, this command may be acceptable. If not, this may be due to a mistake in the original Sierra source code. Either way, if you intend to make this game available on MSDOS platform, you will need to carefully examine the decompiled source code, remove the reference to this command, and adjust your code as necessary before recompiling.

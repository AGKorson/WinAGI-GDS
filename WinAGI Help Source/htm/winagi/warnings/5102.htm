Logic Compiler Warning 5102

Object will never catch ego if DISTANCE <= 1

AGI uses the DISTANCE value of the follow.ego command to determine when to set the 'caught' flag. If DISTANCE is <= 1, the object will never reach ego due to object collision. If the object is set to ignore objects (by using the ignore.objs command), it will be able to catch ego even if DISTANCE is <= 1.

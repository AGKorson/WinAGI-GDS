Logic Compiler Warning 5075

'<command>' is not a supported command for this game's interpreter version

Depending on which interpreter version you intend to use to run your game, some commands may not be available. WinAGI uses the Target Interpreter Version property during compilation to determine if a command will be available at runtime. This warning means you are trying to use an AGI command that is not available for the Interpreter Version your game is targeted for. Running your game with that specified version interpreter file will crash if it tries to use the command noted by this warning.

You should either remove the unsupported command, or adjust your game's Target Interpreter Version to include the command.

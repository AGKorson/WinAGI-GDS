Logic Compiler Warning 7012

'<type>' token found when expecting a command; the token will be ignored

Only applicable when using Sierra syntax.

To maintain compatibility with Sierra's logic compiler CG.EXE, WinAGI ignores '(', ')', and '{' characters when an action command, preprocessor token or keyword is expected.

Even though no error is raised, you should still remove the extra characters, as it is bad coding practice.

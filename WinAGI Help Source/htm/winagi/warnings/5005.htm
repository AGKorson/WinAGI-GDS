Logic Compiler Warning 5005

Undefined inventory item for argument <number>

An invalid inventory item number occurs when the index of the inventory item argument marker exceeds the number of items in the OBJECT file. For example, if there are 25 items in the object file, 'i26' is an undefined inventory item. Although you can successfully compile a logic with an invalid inventory item number in a command, AGI raises trappable error #23. It is possible to recover, but that depends on the game programmer writing good error handling code.

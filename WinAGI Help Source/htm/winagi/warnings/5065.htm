Logic Compiler Warning 5065

set.key parameters are not a valid ASCII code or KEY code

AGI interprets the parameters passed to the set.key command as either an ASCII code or a KEY code. If passing an ASCII code, the KEY code must be zero. If passing a KEY code, the ASCII code must be zero. If both values are non-zero, AGI will assign the controller, but the controller will never activate. (The one exception is for joystick button assignments; in that case, the first argument is '1', and the second will be 1-4 depending on which button press is being assigned.)

See the Keyboard Controllers topic for additional information.

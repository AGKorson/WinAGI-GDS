Resource Warning RW04

Error while loading WAG file; some properties not loaded. (Error <errornumber>: <errortext>)

When WinAGI opens a game, it reads data from the WAG file to set various parameters for the development environment. This error indicates data in the WAG file was not formatted correctly, and WinAGI was unable to determine the cause. It most likely means either the file is corrupted, or has been manually edited in a manner that created errors in some of the entries.

Closing the game should clear the problem. You can also manually edit the file and make repairs if desired.

Layout Settings

 

Editor Options

Use Layout Editor For New Games

If you never plan to use the layout editor, select this option. This disables the Layout Editor by default when a new game is created. You can always re-enable the Layout Editor later for any game by going to the Game Properties dialog.

Synchronize with Resource List

Select this option to have the layout editor synchronized with the resource tree. When synchronized, selecting a room's logic in the resource tree will automatically scroll the layout editor surface so the room is visible in the editor window. This makes it easier to navigate large, complex layouts quickly.

Snap to Grid

When this option is enabled, objects will automatically snap to the nearest minor grid location. This option works whether the grid is being displayed or not. This helps align objects such as rooms, comments, and transfer points.

Show Grid Lines

When this option is enabled, the layout editor will display dotted lines on the drawing surface to show minor and major grid lines. The major grid lines are slightly darker than the minor grid lines. If you do not want to see the grid lines, disable this option.

Display Picture Thumbnails

When enabled, adding new rooms or showing existing logics as rooms will include a thumbnail image of the room's associated picture (the picture resource with the same number as this room). If the picture does not exist, the room will remain blank.

This option only affects the default setting, when new rooms are added. You can always toggle each room's picture property separately on the layout.

Default Scale

You can set the default scale factor to use when extracting a layout. The minimum is one; the maximum is nine.

Grid Spacing

To aid in the alignment of objects, you can use grid lines. grid lines are displayed as light dotted lines placed at regular intervals. Major grid lines are slightly darker and have a spacing that is a multiple of minor grid spacing. When using the Snap to Grid option, objects will snap to the nearest minor grid line, even if the grid is not visible.

Minor grid spacing can be set to increments of 0.05, with a minimum grid value of 0.05 and a maximum of 1.00. Major grid spacing must be an integral multiple of the minor grid spacing.

 

Object Color Settings

The colors used to draw layout editor elements are fully customizable.

Element

Select an element from the list to change its color. Elements include the fill attribute and outline attribute for each object type (Room, Transfer Point, Comment, Error Point and Exits).

Color

The current color of the selected element is shown in the color box. Click the browse button to change the color.

Layout Preview

A sample layout that shows how all elements will look with the selected colors. The preview is updated each time the value of the layout property is changed.

v0: Current Room Number

The current room number is set every time the new.room or new.room.v command is called. v0 is set to the value passed. Typically, this value is then used to load and the picture background. There may also be times where logic code needs to know the number of the current room.

Attempting to use this variable for other purposes is not recommended, because AGI will overwrite it every time the current room is changed.

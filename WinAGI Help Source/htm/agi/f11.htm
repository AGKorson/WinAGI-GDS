f11: Noise Channel Available

When AGI initializes, it determines what the sound capabilities for the system it is running on. Systems other than the IBM PC had better sound support, and could play multiple channels of sound. AGI sets the value of f11 to TRUE if the system supports the noise channel. For IBM PC, the noise channel is not available, so f11 is reset to FALSE. For Tandy, Amiga, Apple IIgs and others with more sound channels, it is set to TRUE. This variable is presumably of use to game programmers so they can provide alternate sounds depending on number of channels supported.

AGI also updates f11 every time the restart.game command is executed.

Once it is initialized, AGI does not use this flag anymore; so it can be used for any purpose without causing problems in AGI. Just be aware that it does get reset every time a restart.game command is executed.

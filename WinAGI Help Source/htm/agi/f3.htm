f3: Ego on Signal Line

AGI sets or clears this flag during every interpreter cycle when updating the position of ego. If any pixel of the baseline of ego is on a signal line (pixels with priority value of 2), this flag is set to TRUE. If no pixels are on a signal line, this flag is set false.

This can be useful to detect when ego is at certain locations in a room without stopping its motion.

Note that if ego is not animated, it does not get updated. So flag f3 won't be modified during interpreter cycles when ego is not animated.

Attempting to use this flag for other purposes is not recommended, because AGI overwrites it in every interpreter cycle when updating ego. If you never use ego in your game, then you could use this flag without any concerns.

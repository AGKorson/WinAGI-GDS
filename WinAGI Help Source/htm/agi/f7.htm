f7: Script Table Entry Disabled

Most commands that involve loading or unloading resources are tracked by AGI by writing to a script table so that games can be properly restored. When f7 is to TRUE, AGI will not write entries to the script table. This means that when a game is restored, any resources loaded/unloaded while f7 is TRUE will not be loaded/unloaded when a game is restored. See the AGI Scripts topic for details of what scripts are and how they work in AGI.

Attempting to use this flag for other purposes is not recommended unless you will not include restore.game commands in your game, because it will interfere with script management, possibly crashing AGI when restarting or restoring games.

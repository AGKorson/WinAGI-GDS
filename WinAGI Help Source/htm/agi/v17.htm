v17: Interpreter Error Code

AGI has very basic, limited built-in error detection. When AGI does detects an error condition, the error number is stored in reserved variable v17, and additional information relevant to the error condition is stored in reserved variable v18 (error information). These variables can then be used by the game programmer to respond to the error. See the Internal Error Detection and Handling topic for additional information.

Unless you are certain that you will NEVER have an error in your game, you should not try to use these variables for other purposes. They will get overwritten if an error does get detected.

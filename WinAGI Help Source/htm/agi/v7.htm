v7: Maximum Score

AGI provides built-in score tracking that programmers can use to provide immediate feedback to players as they progress through a game. When a player does something important, the programmer can adjust the value of reserved variable v3 (current score) to update the player's current score. To provide feedback on overall progress, AGI uses reserved variable v7 as the player's maximum score.

If the status line is visible, the maximum score, along with the current score, is displayed on the status line in the format "Score: xx of yy" where xx is v3 (current score) and yy is v7 (maximum score).

Maximum score is usually set by the programmer when a game is first loaded, but it can be changed at any time if desired. Unlike changes to v3 (current score), changing v7 (maximum score) does not automatically update the status line.

This variable can be used for any purpose, just keep in mind that its value will be displayed on the status bar.

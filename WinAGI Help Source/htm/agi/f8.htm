f8: Joystick Double-Click Enabled

When joystick input is enabled, AGI uses flag f8 to enable or disable 'double-clicks' of the two joystick buttons. When f8 is set to TRUE, AGI will use the delay set by reserved variable v15 (double-click speed) as the delay between button clicks which should be treated as a double-click. A controller can then be assigned to the double-click action. If f8 is reset to FALSE, then all clicks of joystick buttons are treated as single clicks regardless of the delay time between button presses. See the Joystick Input topic for additional details.

If you never intend to allow users to a joystick, you can use this flag for any purpose without affecting AGI.

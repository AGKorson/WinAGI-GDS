v23: Sound Attenuation

For Tandy systems, this variable is used as a global volume control. However, the value acts as an attenuation, not volume level. This means a value of zero attenuation plays sounds at maximum value, and a value of 15 attenuation mutes all sounds. If the value is set higher than 15, AGI will use only the lower nibble, or in other words, the attenuation is set to v26 mod 16.

On the Tandy, the value of v23 is added to each note's attenuation value to determine the total attenuation on a note by note basis. If the combined attenuation is greater than or equal to 15, the note is muted.

On PC systems, this variable is not used by AGI. Sounds will play at regular volume through the speaker regardless of the value of v23.

This variable was not reserved by AGI and had no effect in version 2.089.

If no sounds will be used in a game, or if the game is made for a PC, this variable can be used for any purpose without causing problems in AGI.

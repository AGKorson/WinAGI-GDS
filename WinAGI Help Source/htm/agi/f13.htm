f13: Inventory Selection Enabled

Flag f13 affects the behavior of the status command. When f13 is set to TRUE, the status command will display all inventory items currently in the player's inventory, and allow the player to use cursor keys to select an object. When f13 is reset to FALSE, the inventory items are displayed only; nothing can be selected.

See the status command topic for additional information.

This flag was not reserved by AGI and had no effect in version 2.089 and version 2.272.

This flag can be used for any purpose without affecting AGI, but just be aware that the value of f13 will affect calls to the status command.

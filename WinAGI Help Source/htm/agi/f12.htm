f12: game restored

AGI sets flag f12 to TRUE whenever the restore.game command is executed. This allows a game programmer to know when a player has restored a game so additional code can be run to deal with the restore.

At the end of every every interpreter cycle flag f12 is reset to FALSE.

Attempting to use this flag for other purposes is not recommended, because AGI overwrites it in every interpreter cycle.

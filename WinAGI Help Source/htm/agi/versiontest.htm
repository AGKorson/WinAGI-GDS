AGI Testing Versions

Sierra's source code for AGI included the option to compile the AGI files for production (versions shipped with the released versions of their games) or for testing, which was a special version of AGI for in-house testing, prior to releasing games. The test versions were able to read resources as individual files; instead of searching for VOL and DIR files at start up, the test version looked for a text file named 'WHERE' that had four MSDOS directory entries that pointed to the location of compiled logics, pictures, views and sounds, for example:

..\testing\
..\pic\
..\view\
..\snd\

The resource load functions were modified in the test version to load individual resources from the appropriate directory.

There are four known versions of the Test interpreters:

    • x.yyy (appears to be similar to 2.411)
    • 2.438
    • 2.441
    • 3.002.097

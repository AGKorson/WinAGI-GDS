Mouse Input

For non-MSDOS platforms, mouse support appears to exist in both versions 2 and 3. The mouse was only used for movement control of the ego character; AGI did not support a 'point-and-click' environment.

In version 3, Sierra added commands that provided some limited control of the mouse interface. These commands include show.mouse, hide.mouse, fence.mouse and mouse.posn. It appears these commands only functioned on the Amiga platform. Mouse support details are currently unknown.

Modern AGI emulators include mouse support as a movement control option. In addition, some versions of the Sierra interpreters have been modified by the AGI community to provide mouse support in MSDOS.

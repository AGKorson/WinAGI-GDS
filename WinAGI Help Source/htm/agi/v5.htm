v5: Object Edge Code

If there are other screen objects in a room besides ego and one of them touches an edge, AGI will store the number of the object in reserved variable v4 (object touching edge), and the edge code in reserved variable v5. The values used in v5 are the same as for reserved variable v2 (ego edge code).

If two or more screen objects reach an edge in the same interpreter cycle, only the screen object with the highest index number will be stored.

Attempting to use these variables for other purposes is not recommended, because AGI overwrites them in each interpreter cycle.

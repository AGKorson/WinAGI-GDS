AGI Versions

There were three main version of the AGI interpreter. AGI Version 1 was used with King's Quest 1 and King's Quest 2 when they were originally released. The graphics were in CGA but apart from that the games looked very much like the later EGA remakes. WinAGI does not support AGI V1, and and this help file does not include any specifications for V1.

The major differences among the versions of AGI fall into essentially four areas: resource storage, logic commands, encryption and compression. The resources themselves did not significantly change, and are essentially interchangeable among all version 2 and version 3 interpreters.

AGI Version 2

Version 2 was the most common among games released by Sierra.

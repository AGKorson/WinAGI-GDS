f16: No Prompt for restart.game Command

Normally, when the restart.game command is called, the player is prompted to press ENTER to restart, or ESC to cancel. If f16 is set to TRUE, then no prompt is displayed, and the game is restarted without any delay.

Upon restoring a game, flag f16 is reset to FALSE as part of the restart re-initialization.

This flag was not reserved by AGI and had no effect in version 2.089, version 2.272 and version 2.211.

Unless you will never use the restart.game command, attempting to use this flag for other purposes is not recommended, because it will interfere with the restart function.

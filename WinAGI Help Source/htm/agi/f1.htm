f1: Ego is Obscured

AGI sets this flag during every interpreter cycle when updating the position of ego. If every pixel of the cel assigned to ego is not drawn because it is either transparent, or is of lower priority than the background picture, then no pixels of ego will be visible. In this case, AGI sets flag f1 to TRUE. If at least one pixel of ego is visible, then AGI sets flag f1 to FALSE.

This can be useful in case you need to have your code respond in a specific way in case ego is hidden.

Note that if ego is not animated, it does not get updated. So flag f1 won't be modified during interpreter cycles when ego is not animated.

Attempting to use this flag for other purposes is not recommended, because AGI overwrites it in every interpreter cycle when updating ego. If you never use ego in your game, then you could use this flag without any concerns.

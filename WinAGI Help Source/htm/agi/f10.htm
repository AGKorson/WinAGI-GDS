f10: Command Tracer Enabled

When f10 is set to TRUE, AGI's command tracer feature is enabled, and calls to the trace.on command will start the command tracer. Pressing the SCROLL LOCK key will also start the command tracer.

When f10 is reset to FALSE, the tracer is disabled and calls to trace.on and pressing the SCROLL LOCK key have no effect.

If the command tracer is active, resetting f10 to FALSE does not exit the tracer; f10 is only checked when calling trace.on or pressing SCROLL LOCK.

See the AGI Command Tracer Topic for additional information.

Attempting to use this flag for other purposes is not recommended, because players can inadvertently enter trace mode by pressing the SCROLL LOCK key while f10 is set to TRUE.

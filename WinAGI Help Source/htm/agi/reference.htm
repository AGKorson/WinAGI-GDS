Adventure Game Interpreter

 

Adventure Game Interpreter, or AGI, is the game engine used by Sierra-On-Line between 1984 and 1989. Classic games such as Kings Quest I - III, Space Quest I - II, and Leisure Suit Larry were built using AGI.

In the mid-to-late 90s, a group of AGI fans were able to successfully disassemble the interpreter and the associated game resources. This allowed anyone to create their own games that use the AGI interpreter.

At the same time, many new tools were created to assist fan programmers with almost all aspects of AGI game creation. WinAGI Game Development System is the latest and most comprehensive tool to be created.

 

This help topic provides detailed reference information about the Adventure Game Interpreter. It is based primarily on a detailed analysis of the 19 fully disassembled MSDOS versions of AGI, from version 2.089 up through version 3.002.149. It also includes some information from the original Adventure Game Interpreter Specifications file, created by Peter Kelly, and updated by Claudio Matsuoka in 1999 as well as information about AGI obtained from several other sources, especially Nick Sonneveld's NAGI source code, the Mega-Tokyo AGI message board, and other Internet websites.

This file provides the most accurate, detailed information and corrects some incorrect information in the original AGI Specifications.

Non-MSDOS platforms, such as Apple, Amiga and others were not included in the disassembled file analysis, and thus are not addressed in this reference file (a few instances where previous information has been learned about these non-MSDOS platforms are included here). Also, versions preceding 2.089 are specifically not included. Recently, some original source code and official Sierra documentation on an earlier version of AGI (most likely 2.000) has been discovered; information from that documentation that is incorporated in this reference file where it is relevant.

Data Types

AGI uses the commands in logic resources to manipulate the various data types to make the game work. Commands are used to load and unload resources, draw background pictures, create and animate objects that appear on the screen, etc. Data such as variables and boolean flags are used by commands to track game events and facilitate complex manipulation of resources.

Internal data types used by AGI include variables, flags, screen objects, strings, player-entered words, and controllers. Game resources (logics, pictures, sounds, views, inventory items, and vocabulary words) represent external data types used by AGI. Messages are a subset of logic data that are also used by AGI.

AGI provides storage space for internally managed data types (variables, flags, screen objects, strings, player-entered words and controllers). Messages are a special type of string data that are associated with and stored externally in logic resources.

Variables

Variables represent numeric data used in AGI games. Note that some variables (0 - 26) are reserved by AGI to manage internal game functions; all others are free to be used by the programmer for storing and manipulating game related data.

Flags

Flags are used to manage game state information; they are boolean data and can represent either 'true' or 'false'. Note that some flags (0 - 16) are reserved by AGI to manage internal game functions; all others are free to be used by the programmer for storing and manipulating game related data.

Screen Objects

Screen objects represent the actual sprites that are displayed on the screen to represent movable in-game objects, such as the player character (ego), other characters, etc.

Strings

AGI makes a small number of strings available for use within AGI. The available options to manipulate them are small, but there are several useful features associated with strings in an AGI game.

Player-entered Words

When a player types a command and presses <ENTER>, AGI parses the input, and makes the words typed available for use by AGI.

Controllers

Controllers are used to provide keyboard shortcuts and menu interactions within an AGI game.

Resources

Details for logic, picture, sound, and view resource data types can be found in their respective topics.

Inventory Items

Inventory items, sometimes called inventory objects, should not be confused with screen objects. Inventory items are things that ego can 'pick up' and carry. AGI provides a list of all available inventory items in the OBJECT file.

Vocabulary Words

All the words that a player might use when typing input to play the game are included in the WORDS.TOK file.

Messages

Messages are an external data type, stored within logic resources.

Reserved Values

Several values among the various data types have special meaning to the interpreter and are considered 'reserved'. In most cases, these values (variables v0-v26, flags f0-16 and f20, screen object o0, string s0) should not be used by a game programmer unless their reserved usage is fully understood.

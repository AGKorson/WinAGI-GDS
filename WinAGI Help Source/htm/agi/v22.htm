v22: Sound Channel Count

When AGI initializes, it determines the sound capability of the system it is running on. Systems other than the IBM PC had better sound support, and could play multiple channels of sound. AGI sets the value of v22 to the number of supported sound channels. For IBM PC, v22 is set to 1, for Tandy, Amiga, Apple IIgs and others with more sound channels, it is set to 3. This variable is presumably of use to game programmers so they can provide alternate sounds depending on number of channels supported.

AGI also updates v22 every time the restart.game command is executed.

In version 2.272, a bug was introduced that set the value of v22 to 3 regardless of computer type. The bug was fixed in version 2.425.

Once it is initialized, AGI does not use this variable anymore; so it can be used for any purpose without causing problems in AGI. Just be aware that it does get reset every time a restart.game command is executed.

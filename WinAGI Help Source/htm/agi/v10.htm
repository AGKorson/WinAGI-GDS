v10: Interpreter Animation Interval

The speed of a game can be adjusted by changing the value of v10. In each interpreter cycle, AGI will delay the start of the next cycle by 1/20 second times value of v10. If v10 is set to zero, then there is no delay, and AGI will cycle as fast as the system will allow.

Typical values of animation interval for Sierra games were 4, 2, 1, and 0 for "slow", "normal", "fast" and "fastest" speed options respectively.

Do not try to use this for any other purpose.

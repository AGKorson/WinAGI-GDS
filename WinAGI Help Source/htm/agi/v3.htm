v3: Current Score

AGI provides built-in score tracking that programmers can use to provide immediate feedback to players as they progress through a game. When a player does something important, the programmer can adjust the value of v3 to update the player's current score. Each interpreter cycle, AGI checks the value of v3; if it has changed, AGI will update the current score display in the status line, if the status line is visible. When AGI updates the status line, the current and maximum score is displayed as "Score: xx of yy" where xx is current score (reserved variable v3) and yy is maximum score (reserved variable v7).

This variable can be used for any purpose, just keep in mind that its value will be displayed on the status bar.

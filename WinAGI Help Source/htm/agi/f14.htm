f14: Menus Enabled

Flag f14 determines whether or not the menu.input command will trigger the display of a game's menu. If flag f14 is FALSE when menu.input is called, nothing will happen. If it is TRUE, then AGI will display the menu at the start of the next interpreter cycle.

When AGI starts, and when the restart.game command is executed, flag f14 is reset to FALSE. You must again set it to TRUE in order to display a menu after restarting.

This flag was not reserved by AGI and had no effect in version 2.089 and version 2.272.

If your game uses menus, attempting to use this flag for other purposes is not recommended, because it will interfere with menu operation. If your game will not use menus, then this flag can be used for any purpose without causing any problems.

v8: Pages of Free Memory

The memory available for game resources in AGI was limited to approximately 57 KBytes due to limitations in hardware and software available at the time AGI was created. To assist game programmers in managing memory and avoid crashing due to exceeding available memory, AGI uses v8 to store the number of 256 byte 'pages' of memory (i.e., v8=free memory/256, rounded down) that are available at any given time. For additional information on memory management in AGI, see the Memory Management topic.

Attempting to use this variable for other purposes is not recommended, because AGI overwrites it every time resources are loaded or unloaded in memory.

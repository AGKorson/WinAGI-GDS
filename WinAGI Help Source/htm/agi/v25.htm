v25: Selected Inventory Item

AGI sets v25 to the index number of the inventory item selected from the inventory screen (when called by the status command with reserved flag f13 (inventory selection enabled) set to TRUE). If no object is selected, (i.e. the player presses ESC), AGI puts a value of 255 in v25. The index number of the selected object can then be used to display a view of the object.

When the status command is called, if v25 value matches an object in inventory, it will be the default selected object.

This variable was not reserved by AGI and had no effect in version 2.089 and version 2.272.

Attempting to use this variable for other purposes is not recommended, unless your game does not use the status command for object selection.

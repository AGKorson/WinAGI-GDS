f20: Force Auto-Loop Selection

NOTE: This flag is only reserved in AGI versions 3.002.98 and above.

Normally, AGI automatically changes the loop for screen objects whenever the direction of the object changes, as long as the number of loops in the view is four or less. Beginning with AGI version 3.002.98, flag f20 can be set to TRUE to force AGI to automatically update the loop even if there are more than four loops in the view.

See the Auto-Loop Feature Topic for additional information.

This flag can be used for any purpose without affecting AGI if using versions before 3.002.98. In versions that use f20, if there are no views with five or more loops assigned to a screen object, f20 can also be used for any purpose without affecting AGI.

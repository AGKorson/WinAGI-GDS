v1: Previous Room Number

When the new.room is called, the current value of v0 is first copied to v1 before v0 is updated with the new room number. This value is not used by AGI internally in any way. It appears to be provided as an aid to programmers when managing the flow of games across multiple rooms.

Attempting to use this variable for other purposes is not recommended, because AGI will overwrite it every time the current room is changed.

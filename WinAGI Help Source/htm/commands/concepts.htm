Programming Concepts

Writing successful code for an AGI game is much more difficult than it may appear. A programmer needs to understand the fundamental operation of AGI.

This help file does not contain any tutorials to assist in learning the nuances of AGI programming. There are some tutorials that are available on the web that may be useful.

Currently, the best way to learn AGI programming is to use WinAGI to examine the code from existing games, both Sierra and fan-based, as well as seeking help from other fan game designers.

While not fully detailed tutorials, this help file does contain some key information on dealing with a few common tasks in AGI. In addition, the command reference topics contain a lot of detailed information on individual commands.

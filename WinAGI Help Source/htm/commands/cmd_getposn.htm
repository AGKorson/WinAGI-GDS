get.posn

The get.posn command copies a screen object's current X and Y position values into variables.

 

Syntax

get.posn(obj oA, var vX, var vY);

Remarks

Variables vX and vY are set to the current coordinates of screen object oA. AGI does not check that object number oA is a valid screen object. If it is not, the returned values will be meaningless.

Example

get.posn(o1, v51, v52);
print("Object position is %v51, %v52");

 

Required Interpreter Version

Available in all AGI versions.

 

Byte-Code Value

39 (0x27)

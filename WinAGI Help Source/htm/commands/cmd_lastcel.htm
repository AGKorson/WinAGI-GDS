last.cel

The last.cel command stores the last cel number of an object's current loop in a variable.

 

Syntax

last.cel(obj oA, var vB);

Remarks

Variable vB is set to the last cel number of screen object oA's current loop. Because the first cel is numbered 0, the last cel will be the cel count - 1.

AGI does not check that object number oA is a valid screen object. If it is not, the returned value will be meaningless.

Example

last.cel(o0, v51);  [ v51 is set to last cel of ego's current loop

 

Required Interpreter Version

Available in all AGI versions.

 

Byte-Code Value

49 (0x31)

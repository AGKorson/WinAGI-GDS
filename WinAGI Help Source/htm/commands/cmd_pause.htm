pause

The pause command pauses the game until the player presses ENTER or ESC key.

 

Syntax

pause();

Remarks

A message box is displayed that says "Game paused. Press Enter to continue." Pressing ENTER or ESC will dismiss the window and the game will continue.

When the game is paused using the pause command, the interpreter's internal clock is also paused. If a sound is playing, it is stopped.

Reserved flag f15 (no pause windows) and reserved variable v21 (window close time) affect the pause window the same as any other message box. This means if you set f15 and then use pause, the message box will be displayed, but the game won't actually pause.

 

Example

pause(); [ game play is paused; internal clock is paused
[ similar print command:
print("      Game paused.\n"
      "Press Enter to continue."); [ game play is paused, but clock continues to run

 

Required Interpreter Version

Available in all AGI versions.

 

Byte-Code Value

136 (0x88)

assignn

The assignn command assigns a numeric value to a variable.

 

Syntax

assignv(var vA, byt B);

vA = B;

Remarks

None.

 

Example

assignv(v50, 75);
v50 = 75;

 

Required Interpreter Version

Available in all AGI versions.

 

Byte-Code Value

3 (0x03)

set.upper.left

The purpose of the set.upper.left command not currently known.

 

Syntax

set.upper.left(byt A, byt B);

Remarks

In MSDOS, this command points to a null function that skips over the passed arguments. According to Sierra's original source code for AGI, this command is only used on Apple systems. Its purpose on the Apple is currently unknown.

 

Example

set.upper.left(0, 0); [ does nothing on MSDOS AGI games

 

Required Interpreter Version

Available in version 2.272 and above.

 

Byte-Code Value

155 (0x9B)

Working with Screen Objects

Screen objects (not to be confused with inventory objects) are what provide the animation in AGI games. They can represent various things such as characters, background animations or still props. Each object has a view assigned to it, and can be placed anywhere on screen and moved around.

The first object, object o0, is reserved by AGI for player control. This object is commonly referred to as "ego". The player controls ego by using the arrow keys.

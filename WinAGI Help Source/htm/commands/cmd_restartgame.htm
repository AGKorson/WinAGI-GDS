restart.game

The restart.game command restarts a game from the beginning.

 

Syntax

restart.game();

Remarks

When the restart.game command is executed, AGI displays a message box asking the player to confirm (press ENTER) or cancel (press ESCAPE) before restarting. Beginning with version 2.425, reserved flag f16 can be set to TRUE to tell AGI to skip the message box and immediately restart without getting player input.

Restarting a game resets AGI and starts the game over from the beginning. See the Restarting Games topic for details on what happens when AGI restarts the game.

 

Example

if (said("restart"))
  {
  restart.game();
  }

 

Required Interpreter Version

Available in all AGI versions.

 

Byte-Code Value

128 (0x80)

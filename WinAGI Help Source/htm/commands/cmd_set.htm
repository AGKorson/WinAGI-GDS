set

The set command sets a flag's value to TRUE.

 

Syntax

set(flg fA);

fA = true;

Remarks

Changing values of reserved flags can have unpredictable results, and should be done carefully.

 

Example

set(f99);
reset(f99);
f99 = true;  [ same as set(f99);
f99 = false; [ same as reset(f99);

 

Required Interpreter Version

Available in all AGI versions.

 

Byte-Code Value

12 (0x0C)

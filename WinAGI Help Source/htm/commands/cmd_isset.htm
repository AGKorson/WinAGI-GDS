isset

The isset command returns the value of the specified flag.

 

Syntax

isset(flg fA)

fA

Remarks

Test commands are only valid in an if statement.

The isset command returns TRUE if flag fA is set, FALSE if it is not.

This statement can be combined with the NOT operator to create a 'flag not set' test.

 

Example

if(isset(f40))
  { 
  print("flag 40 is set (TRUE)");
  }

if(f40)  [ alternate syntax
  {
  print("flag 40 is set (TRUE)");
  }

 

Required Interpreter Version

Available in all AGI versions.

 

Byte-Code Value

7 (0x07)

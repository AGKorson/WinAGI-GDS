status.line.off

The status.line.off command disables and hides the status line.

 

Syntax

status.line.off();

Remarks

The status line, which displays the current score and sound on/off status, is immediately removed from the display, and is disabled to prevent it from showing again. It will not be redisplayed until a status.line.on command is executed.

 

Example

if(said("hide", "status"))
  {
  status.line.off();
  }

 

Required Interpreter Version

Available in all AGI versions.

 

Byte-Code Value

113 (0x71)

Working with Pictures

The steps necessary to get a picture background displayed on the screen should always be done in this order:

    • load the picture resource (load.pic);
    • draw the picture to memory (draw.pic);
    • discard the picture resource (discard.pic);
    • other commands that manipulate the image, such as overlay.pic;
    • finally, show the visual screen on the actual screen (show.pic).

The arguments for the picture commands are always variables; there are no equivalent commands that take numbers. Usually, these commands are used to draw the current room, which is stored in reserved variable v0.

If you want to draw a picture other than the current room, store the resource number in another variable, then use the picture commands on that variable.

Use the overlay.pic command if you want to draw a picture without first clearing the existing screen. An excellent example of using the overlay.pic command can be found in Gold Rush!, where the overlay.pic command is used to generate random forest scenes.

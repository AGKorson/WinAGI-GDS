get.dir

The get.dir command stores an object's direction in a variable.

 

Syntax

get.dir(obj oA, var vDIR);

Remarks

Variable vDIR is assigned the direction of screen object oA.

AGI does not check that object number oA is a valid screen object. If it is not, the returned value will be meaningless.

Example

get.dir(o12, v50);
[ v50 now equals the direction of o12

 

Required Interpreter Version

Available in all AGI versions.

 

Byte-Code Value

87 (0x57)

get.priority

The get.priority command stores the current priority of an object in a variable.

 

Syntax

get.priority(obj oA, var vPRI);

Remarks

Variable vPRI is set to the priority of screen object oA. AGI does not check that object number oA is a valid screen object. If it is not, the returned value will be meaningless.

Example

get.priority(o1, v50);
print("Object priority is %v50);

 

Required Interpreter Version

Available in all AGI versions.

 

Byte-Code Value

57 (0x39)

accept.input

The accept.input command displays the input line and allows a player to enter text input.

 

Syntax

accept.input();

Remarks

The input line is hidden by default when AGI starts, and needs to be enabled with accept.input. When called, accept.input immediately draws the input line on the row specified by the most recent configure.screen command, using the current display prompt and cursor character. When the input line is displayed, any text that was on the line at the time it was hidden will be displayed as well.

Use the prevent.input command to hide the input line.

 

Example

prevent.input(); [ input prompt is hidden, and player cannot enter text input 
...
accept.input(); [ input prompt is restored/redisplayed, including any text
                       [ that was on it when it was hidden

 

Required Interpreter Version

Available in all AGI versions.

 

Byte-Code Value

120 (0x78)

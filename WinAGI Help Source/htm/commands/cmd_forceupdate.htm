force.update

The force.update command redraws all screen objects immediately, without waiting until the start of the next interpreter cycle.

 

Syntax

force.update(obj oA);

Remarks

Screen objects are normally all redrawn at the end of each interpreter cycle. The force.update command tells AGI to redraw all screen objects immediately. Screen objects will also be redrawn at the end of the next cycle as usual.

Although the command implies that only screen object oA is updated, all screen objects are redrawn when a force.update command is executed regardless of the object number passed. The argument value is ignored. You can even use an invalid object number with no bad effect.

This command does not change an object's update state. If an object has been stopped with the stop.update command, use the start.update command to restart it.

Example

force.update(o255);  [ redraw all objects; number used in this command is ignored

 

Required Interpreter Version

Available in all AGI versions.

 

Byte-Code Value

60 (0x3C)

stop.motion

The stop.motion command stops an object from moving.

 

Syntax

stop.motion(obj oA);

Remarks

This command sets screen object oA's direction to 0, which causes it to stop moving, and sets its movement mode to 'normal'. If the object being stopped is ego (o0) it also switches ego to program control.

This command is functionally equivalent to using set.dir command with an argument value of 0, and then a normal.motion command. (And also a program.control command, if the object is ego).

AGI does not check that object number oA is a valid screen object. If it is not, this command will overwrite other data on the heap, which may cause unexpected results, including possibly crashing AGI.

 

Example

move.obj(o3, 30, 140); o3 is moving, in 'move object' mode
stop.motion(o3); o3 stops moving; mode is back to 'normal'

[ equivalent code to stop o3
v51 = 0;
set.dir(o3, v51);
normal.motion(o3);

 

Required Interpreter Version

Available in all AGI versions.

 

Byte-Code Value

77 (0x4D)

print

The print command displays a message in a window.

 

Syntax

print(msg mMSG);

print("message text");

Remarks

The text of message mMSG is displayed in a bordered window. The background color is always white, and the foreground color is always black.

AGI will center the window on the screen, and if the message is more than 30 characters long, AGI will automatically add line breaks.

Reserved flag f15 (no pause windows) and reserved variable v21 (window close time) determine how the print command affects game play:

    • f15 reset to FALSE and v21 == 0 (the default condition): displayed windows (including those shown by the print command) pause the game cycle, and remain until the player presses ENTER or ESC key. Although the game cycle is paused, the internal clock does not stop, and clock variables v11, v12, v13, v14 will continue to update.
    • f15 reset to FALSE and v21 > 0: the window will also pause the game cycle, and will be dismissed when the player presses ENTER or ESC key, or after v21 * 0.5 seconds, whichever comes first.
    • f15 is set to TRUE (v21 is ignored): the game cycle is NOT paused, and the window remains open indefinitely. The message window stays open and game play continues until a close.window command is called to dismiss the message, or another message is displayed.

After every time a window is displayed (including by using the print command), if f15 is TRUE, it is reset to FALSE and variable v21 is not modified. If f15 is FALSE, variable v21 is reset to 0.

(In version 2.089 only, reserved flag f15 is not used; message windows will never print without pausing.)

Messages can contain special format codes to enhance the output. See the Message Format Codes topic for details on available message codes.

 

Example

[ format code examples
v50 = 99;
print("%v50 bottles of beer on the wall"); 
[ will show on screen as  "99 bottles of beer on the wall"

#message 3 " again."
print("You got killed%m3");
[ will show on screen as "You got killed again."

s1 = "20 years"
print("You are %s1 old.");
[ will show on screen as "You are 20 years old."

[ player enters "eat dirt"
print("Why eat %w2?");
[ will show on screen as "Why eat dirt?"

 

Required Interpreter Version

Available in all AGI versions.

 

Byte-Code Value

101 (0x65)

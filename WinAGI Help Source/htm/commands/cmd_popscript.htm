pop.script

The pop.script command restores the script table pointer.

 

Syntax

pop.script();

Remarks

This command should only be called after push.script, which saves the current script pointer.

pop.script restores the script pointer. Subsequent script entries will be added at this location. See the AGI Scripts topic for additional information.

Example

push.script(); 
...  [ some activities that add script entries
pop.script();  [ script pointer is reset

 

Required Interpreter Version

Available in version 2.903 and above.

 

Byte-Code Value

172 (0xAC)

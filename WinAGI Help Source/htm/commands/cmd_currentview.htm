current.view

The current.view command stores the number of the view currently assigned to an object in a variable.

 

Syntax

current.view(obj oA, var vB);

Remarks

Variable vB is set to the view number for object oA.

AGI does not check that object number oA is a valid screen object. If it is not, the returned value is meaningless.

Example

current.view(o0, v52);  [ v52 now equals the view number assigned to ego

 

Required Interpreter Version

Available in all AGI versions.

 

Byte-Code Value

52 (0x34)
